# 暗黑3式系统框架设计

> 基于暗黑破坏神III核心系统循环，适配自动战斗挂机游戏的完整设计框架。
> 局内战斗改为自动化表演，局外与长线体验对标暗黑3。

---

## 一、设计哲学

暗黑3 的玩家乐趣 = **刷怪手感 + 掉落多巴胺 + Build构筑 + 无限进阶**

本项目需要把"刷怪手感"替换为"自动战斗表演观赏"，把玩家的主动参与集中在后三项上。

```
暗黑3玩家:  操作战斗 → 看掉落 → 换装备 → 调Build → 冲更高层
本项目玩家:  观看战斗 → 看掉落 → 换装备 → 调Build → 冲更高层
             ↑                                        ↑
        自动化+表演感                            决策感不能丢
```

## 三、战斗系统 — 从"操作"到"表演"

### 3.1 设计原则

玩家不操作战斗，但战斗**必须好看、必须可读**，让玩家能"看懂"自己的 Build 在起作用。

### 3.2 自动战斗表演层

| 元素 | 暗黑3 | 本项目适配 |
|------|-------|-----------|
| 技能释放 | 玩家手动按 | AI按优先级自动释放，屏幕显示技能名+特效 |
| 走位闪避 | 玩家手动走 | 角色自动移动，遇大招自动闪避(有概率) |
| 怪物密度 | 玩家选择路线 | 系统根据节点配置生成波次 |
| 战斗节奏 | 操作决定 | 由攻速/技能CD决定，玩家可通过装备影响 |

### 3.3 战斗可读性设计

战斗画面需要让玩家"看见"Build的效果：

| 触发事件 | 视觉反馈 |
|---------|---------|
| 套装特效触发 | 全屏闪光 + 文字提示 "风眼·四件效果触发" |
| 暴击 | 数字放大 + 颜色变化 |
| 传奇特效触发 | 专属动画 + 音效 |
| 濒死触发 | 屏幕边缘泛红 → 防御特效激活 |
| 秒杀 | 快速清屏表演 → 满足感 |

### 3.4 战斗速度控制

| 节点类型 | 战斗时长 | 侧重 |
|---------|---------|------|
| 普通节点 | 5-10秒/波 | 效率和掉落数量 |
| 精英节点 | 15-20秒 | 更多特效触发，掉落品质更高 |
| Boss节点 | 30-60秒 | 全特效展示，可能掉落套装/传奇 |
| 秘境层 | 根据层数动态调节 | 卡关时战斗明显变慢→提示需要更好装备 |

## 四、掉落系统 — 多巴胺引擎

### 4.1 品质分层

| 暗黑3品质 | 本项目名称 | 颜色 | 掉落表现 | 出现频率 |
|-----------|----------|------|----------|---------|
| Normal | 凡品 | 灰白 | 直接显示为资源 | 极高 |
| Magic | 灵品 | 蓝 | 小抛物线落地 | 高 |
| Rare | 玄品 | 黄 | 中抛物线+光柱 | 中 |
| Legendary | 真意 | 橙 | 大抛物线+全屏闪+音效 | 低 |
| Set | 道统 | 绿 | 大抛物线+专属纹路+音效 | 极低 |
| Ancient | 远古真意 | 橙框金光 | 同真意+额外金色粒子环绕 | 稀有 |
| Primal | 天道真意 | 红光 | 全屏红光+震屏+独特音效 | 极稀有 |

### 4.2 掉落节奏设计

| 品质 | 每分钟频率 | 玩家行为 |
|------|-----------|---------|
| 凡品/灵品 | 10-20个 | 自动分解为材料，不打扰玩家 |
| 玄品 | 2-5个 | 快速扫一眼，多数分解 |
| 真意 | 每5-10分钟1个 | 停下来看一眼 |
| 道统 | 每30-60分钟1个 | 兴奋点 |
| 远古真意 | 每2-4小时1个 | 今日高光 |
| 天道真意 | 每天0-1个 | 分享级事件 |

**关键**：低品质物品要自动处理（自动分解机制），玩家只需要关注稀有及以上的掉落决策。

## 五、装备系统 — Build 构筑核心

### 5.1 装备槽位

| 部位 | 数量 | 可出词缀数 | 说明 |
|------|------|-----------|------|
| 兵器 | 1 | 4-6 | 决定基础伤害和攻击方式 |
| 头盔 | 1 | 3-5 | 常见CDR/资源类词缀 |
| 铠甲 | 1 | 3-5 | 防御为主 |
| 护手 | 1 | 3-5 | 暴击/攻速 |
| 腿甲 | 1 | 3-5 | 混合 |
| 靴子 | 1 | 3-5 | 移速/抗性 |
| 佩饰 | 2 | 3-4 | 传奇宝石槽 + 词缀 |
| 腰带 | 1 | 3-4 | 特殊机制类 |

共 **9 个槽位**，足够构成有深度的 Build 但不至于太复杂。

### 5.2 词缀池设计（多乘区结构）

**主属性词缀（加区A）：**

| 属性 | 效果 |
|------|------|
| 力道 +X | 每点+1%物理伤害, +1护甲 |
| 灵慧 +X | 每点+1%元素伤害, +1全抗 |
| 根骨 +X | 每点+30生命值 |

**双暴词缀（乘区B）：**

| 属性 | 范围 |
|------|------|
| 会心率 | 1%-10% |
| 会心伤害 | 10%-100% |

乘区公式：`1 + (会心率 × 会心伤害)`

**技能伤害词缀（乘区C）：**

| 属性 | 范围 |
|------|------|
| 御风刀伤害 | +15%-30% |
| 血劫手伤害 | +15%-30% |
| 五雷符伤害 | +15%-30% |

**元素伤害词缀（乘区D）：**

| 属性 | 范围 |
|------|------|
| 风属性伤害 | +10%-20% |
| 火属性伤害 | +10%-20% |
| 雷属性伤害 | +10%-20% |

**传奇/套装特效（独立乘区E）：**

| 特效 | 说明 |
|------|------|
| "风眼6件: 御风刀伤害 +8000%" | 独立乘算 |
| "血劫4件: 每流血层+200%伤害" | 独立乘算 |

### 5.3 词缀生成规则

- 词缀等级 (alvl) 与物品等级 (ilvl) 挂钩
- 同一物品上不能出现重复词缀（无放回抽样）
- 前缀/后缀无固定搭配限制
- 每条词缀在一个范围内随机取值

## 六、数值系统 — 多乘区伤害公式

### 6.1 完整伤害公式

```
最终伤害 = 武器基础伤害
         × (1 + 主属性/100)           ← 乘区A: 主属性
         × (1 + 会心率 × 会心伤害)     ← 乘区B: 双暴
         × (1 + 技能伤害%/100)         ← 乘区C: 技能伤害
         × (1 + 元素伤害%/100)         ← 乘区D: 元素伤害
         × (1 + 套装特效%/100)         ← 乘区E: 套装(巨额)
         × (1 + 传奇特效%/100)         ← 乘区F: 传奇特效
         × (1 + 精英增伤%/100)         ← 乘区G: 仅对精英/Boss
```

### 6.2 加区 vs 乘区

| 关系 | 说明 | 举例 |
|------|------|------|
| **加区 (Additive)** | 同一乘区内的数值相加 | 两件 +20% 火焰伤害 = 共 +40% |
| **乘区 (Multiplicative)** | 不同乘区之间相乘 | +100% 技能伤 × +100% 套装伤 = 4倍(非3倍) |

**设计意义**：同类词缀堆越多，单条收益递减（加区内部）；不同类词缀组合，收益爆炸增长（跨乘区相乘）。自然引导玩家追求**均衡搭配**而非单一堆叠。

## 七、套装系统 — "道统"

### 7.1 道统设计

每个"道统"（套装）对应一个核心流派，由 6 件装备组成：

| 道统名 | 对应流派 | 2件效果 | 4件效果 | 6件效果 |
|--------|---------|---------|---------|---------|
| **御风道统** | 御风刀 | 御风刀攻速+30% | 御风刀每命中回复1%灵力 | 御风刀伤害+8000% |
| **血劫道统** | 血劫手 | 暴击使敌人流血5秒 | 流血敌人受伤+200% | 每层流血+500%伤害，上限10层 |
| **五雷道统** | 五雷符 | 雷击有30%概率连锁 | 连锁雷击伤害不衰减 | 五雷符伤害+6000%，连锁上限+5 |
| **铁壁道统** | 坦克/反伤 | 受击回复2%生命 | 格挡时反弹200%伤害 | 护甲的50%转化为额外伤害 |

### 7.2 套装获取

| 途径 | 说明 |
|------|------|
| 推图节点掉落 | 随机套装部件(低概率) |
| Boss击杀 | 指定道统部件(较高概率) |
| 秘境宝箱 | 高品质套装部件 |
| 炼器转换 | 同道统内部件互换(对标暗黑3魔盒) |

### 7.3 辅助机制

- "华戒"类饰品 → 减少1件套装需求 → 允许混搭两套道统

## 八、炼魂鼎 — 对标卡奈魔盒

### 8.1 功能列表

| 功能 | 暗黑3对应 | 说明 | 消耗 |
|------|-----------|------|------|
| **萃取真意** | 萃取传奇特效 | 分解真意装备，提取橙字特效到"真意图录"，可从兵器/防具/佩饰各激活1个 | 灵核×5 + 真意装备 |
| **锻玄升真** | 升级稀有→传奇 | 将玄品升级为随机同类真意 | 祠灰×25 + 玄品 |
| **回炉重铸** | 重铸传奇 | 重置真意装备属性，有概率变远古 | 各类材料×大量 |
| **道统互转** | 转换套装 | 同道统内部件互换 | 灵核×2 |
| **悟道精炼** | 附魔 | 重随装备上的1条词缀(锁定其他) | 祠灰×10 |

### 8.2 真意图录（对标卡奈特效栏）

```
┌─────────────────────────────────┐
│           真意图录               │
├───────────┬─────────┬───────────┤
│  兵器栏    │  防具栏   │  佩饰栏   │
│           │          │          │
│ ◆ 风眼    │ ◆ 铁壁   │ ◆ 连锁    │
│  (已激活)  │ (已激活)  │ (已激活)  │
│           │          │          │
│ ○ 血刃    │ ○ 回春   │ ○ 聚灵    │
│ ○ 雷引    │ ○ 冰甲   │ ○ 暴怒    │
│ ○ ...     │ ○ ...   │ ○ ...    │
└───────────┴─────────┴───────────┘
  各栏选1个激活 = 额外3个传奇特效
```

**设计意义**：
- 让已经不穿的传奇装备有了"分解价值"，而非纯粹卖钱
- 额外3个特效槽大幅增加 Build 深度
- 收集所有真意特效成为长线目标

## 九、秘境系统 — 无限进阶

### 9.1 试炼秘境

| 参数 | 值 | 说明 |
|------|-----|------|
| 起始层 | 1 | 无上限 |
| 每层怪物血量 | +17% | 指数增长 |
| 每层怪物伤害 | +5% | 缓增 |
| 限时 | 3分钟/层 | 击杀Boss |
| 钥匙来源 | 普通推图获取 | 保证资源循环 |

### 9.2 通关奖励

| 奖励 | 说明 |
|------|------|
| 经验 | 巅峰等级经验 |
| 传奇宝石升级 | 每次+1级，成功率随级差递减 |
| 每10层里程碑 | 额外奖励宝箱 |
| 首通奖励 | 10/20/30/50/70/100层各有特殊奖励 |

### 9.3 天道真意解锁

- 单人通过试炼秘境 70层 → 全局解锁天道品质掉落
- 首次通关必定掉落1件天道真意（满值词缀）
- 此后所有内容均有极低概率掉落天道

### 9.4 数值缩放参考

| 维度 | 每层缩放 | 每30层 |
|------|----------|--------|
| 怪物血量 | +17% | 约×70倍 |
| 玩家所需韧性 | +2.34% (70层起) | ×2倍 |
| 掉落品质 | 逐步提升远古/天道概率 | — |

## 十一、轮回系统 — 对标赛季重置

适配为修仙题材的"轮回"系统。

### 11.1 轮回机制

| 项目 | 说明 |
|------|------|
| 触发条件 | 玩家主动选择"入轮回"(非强制) |
| 角色等级 | 归1 |
| 装备 | 全部封存到"前世宝库"(不删除) |
| 巅峰等级 | 保留(跨轮回积累) |
| 试炼秘境 | 进度重置 |

### 11.2 轮回奖励

| 奖励类型 | 说明 |
|---------|------|
| 永久属性加成 | 每轮回+2%全属性 |
| 前世宝库 | 每轮回可取出1件前世装备(降级使用) |
| 专属轮回道统 | 仅在轮回模式中掉落的套装 |
| 轮回成就 | 外观/称号/特效 |

### 11.3 设计意义

- 让玩家重新体验"从零到毕业"的成长快感
- 轮回次数提供永久增强 → 每轮回比上轮回更快毕业
- 前世宝库创造跨轮回连续性 → 不是纯粹的归零

## 十二、挂机适配的关键差异

### 12.1 主动 vs 挂机对照

| 维度 | 暗黑3(主动) | 本项目(挂机)适配 |
|------|------------|-----------------|
| 战斗参与 | 玩家操作 | AI自动 + 表演观赏 |
| 装备决策 | 实时切换 | 离线回来集中处理 |
| 掉落处理 | 逐个拾取鉴定 | 低品质自动处理，高品质弹窗通知 |
| 难度选择 | 随时切换 | 推图卡关→系统建议"回退低层刷装备" |
| 离线收益 | 无 | 离线按效率产出材料+低品质装备 |
| Build实验 | 随时换装测试 | 提供"模拟战斗"功能预览DPS变化 |

### 12.2 离线掉落的特殊设计

```
离线期间:
  - 正常计算战斗轮次和掉落
  - 凡品/灵品 → 自动分解为材料(上线时汇总显示)
  - 玄品 → 存入"待鉴定"队列(上线后手动查看)
  - 真意及以上 → 存入"闭关所得·高光"(上线后逐个展示掉落动画)

上线时:
  "闭关所得"弹窗 → 先显示总收益 → 再逐个播放高品质掉落的表演动画
  让离线玩家也能获得"开箱"般的多巴胺体验
```
