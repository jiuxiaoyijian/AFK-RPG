## 协作约定

1. **新建文档**：从 `_模板/` 复制对应模板，填写后放入对应目录
2. **命名规则**：使用中文语义名，不使用编号前缀（编号仅用于顶级目录）
3. **大文档拆分**：超过 500 行的文档应拆分为子目录（如 `01_系统设计/装备系统/`）
4. **变更记录**：重大设计变更须同步更新 `00_项目总纲/变更日志.md`
5. **Review 流程**：文档修改通过 PR 提交，至少一人 review 后合并
