# 世界观与题材包装

> 本项目以"武侠江湖"为题材包装。玩家扮演一名新入门的门派弟子，在自动化的江湖历练中成长为绝世高手。
> 命名规范参见 `13_武侠命名映射表.md`。

---

## 一、题材方向

**武侠江湖** — 非修仙飞升，非玄幻爽文。强调：
- 门派师徒、江湖恩怨、武学传承
- 刀光剑影而非法术飞天
- 凡人通过修炼武艺达到极致，而非成仙成神
- 江湖有规则、有势力、有灰色地带

**不使用**：仙侠、修真、飞升、法宝、灵气、渡劫等概念。

---

## 二、一句话产品定位

> **一柄刀，一身武学，在江湖的刀光剑影中，走出属于自己的武道。**

Steam 页面方向：武侠挂机 RPG / 自动战斗 + 装备驱动 + Build 构筑

---

## 三、世界观基础设定

### 3.1 世界背景

天下武林，门派林立。青云门是中原正派之一，百年前曾鼎盛一时，如今虽仍有声望，但实力已不如往昔。近年来江湖暗流涌动，各方势力蠢蠢欲动，青云门急需新鲜血液。

### 3.2 玩家身份

玩家是一名刚被青云门收入的**外门弟子**。出身平凡，没有显赫背景，一切从零开始。

- **初始状态**：只会基础刀法/拳法，装备粗陋
- **成长路径**：外门弟子 → 内门弟子 → 核心弟子 → 长老 → 宗师
- **动力来源**：在历练中获取武学、兵器、秘传，逐步构筑自己的武道

### 3.3 青云门设定

| 要素 | 设定 |
|------|------|
| 门派名 | 青云门 |
| 位置 | 青云山，中原腹地 |
| 特色 | 兼容并蓄，三大武学流派并存 |
| 掌门 | （NPC，暂不深入） |
| 门派设施 | 练武场(战斗)、百炼坊(锻造)、藏书阁(武学参悟)、试剑台(秘境入口) |

### 3.4 江湖势力

| 势力 | 说明 | 出现章节 |
|------|------|---------|
| 青云门 | 玩家所属正派门派 | 全局 |
| 黑风寨 | 山匪集团，盘踞青云山外 | 第一章 |
| 落雁残部 | 已覆灭门派的残余弟子，占据落雁谷 | 第二章 |
| 铁锁帮 | 控制铁锁镇的江湖帮派 | 第三章 |
| 雪刀门 | 北方剑道门派，亦敌亦友 | 第四章 |
| 剑冢守卫 | 传说中的绝世高手隐居之地 | 第五章 |

## 四、三大流派映射

青云门兼容三大武学流派，弟子可任选其一作为主修方向：

| 流派 | 武器 | 战斗风格 | 一句话 |
|------|------|---------|--------|
| **御风刀** | 刀/斧 | 旋风范围清场 | "刀随风起，万敌俱灭" |
| **血劫手** | 刀/枪 | 流血DOT，低血处决 | "一寸血，一寸伤，血尽人亡" |
| **五雷掌** | 法球/杖 | 连锁弹射雷击 | "掌心惊雷，百步之内皆为雷池" |

## 五、章节世界观

### 第一章：青云山外

- **地形**：山脚村落、荒野小道、山匪营寨
- **故事**：弟子首次下山历练，清除盘踞山脚的黑风寨匪徒
- **Boss**：铁面虬髯客 — 黑风寨寨主，力大无穷，铁甲护身
- **视觉**：青绿山林、泥土小路、简陋木寨

### 第二章：落雁谷

- **地形**：废弃门派遗址、断壁残垣、枯井密道
- **故事**：落雁派覆灭后，残余弟子占据遗址为非作歹，青云门派弟子前往清剿
- **Boss**：枯骨剑客 — 落雁派最后的高手，剑法诡异，身如枯骨
- **视觉**：残破建筑、杂草丛生、阴郁色调

### 第三章：铁锁镇（P2规划）

- **地形**：江湖重镇、市集、地下赌场、码头
- **故事**：铁锁帮控制的商贸重镇，暗中勾结邪派
- **Boss**：铁锁帮主·银钩老人 — 钩法独步，狡诈多端
- **视觉**：热闹市集与阴暗地下的对比

### 第四章：雪刀岭（P3规划）

- **地形**：北方雪原、冰崖、雪中古庙
- **故事**：雪刀门与青云门的武学交流，暗藏危机
- **Boss**：雪刀门少主·寒霜 — 刀法凌厉如寒风
- **视觉**：白雪皑皑、冷色调、刀光如雪

### 第五章：剑冢（远期规划）

- **地形**：传说中的剑冢，埋葬历代绝世名剑
- **故事**：终局挑战，面对剑冢中的传说级守卫
- **Boss**：无名剑灵 — 剑冢中诞生的绝世存在
- **视觉**：古朴厚重、剑气纵横

| 品质 | 叙事包装 | 掉落文案示例 |

## 七、UI与美术风格指引

### 7.1 整体风格

- **水墨武侠风**：淡雅底色 + 笔墨质感 + 武侠气韵
- **配色基调**：深灰/墨色底 + 金色/朱红点缀
- **字体**：使用思源宋体或类似有笔锋感的中文字体

### 7.2 界面元素

| 元素 | 风格 |
|------|------|
| 主界面底色 | 深色水墨纹理 |
| 按钮 | 卷轴/木牌造型 |
| 品质边框 | 简约描边 + 品质色 |
| 装备图标 | 武侠兵器/护具简笔画风 |
| 技能图标 | 招式动态感图标 |

### 7.3 战斗画面

| 元素 | 表现 |
|------|------|
| 角色 | 武侠装束，持兵器 |
| 普攻 | 刀光/拳风 |
| 技能特效 | 御风刀=旋风，血劫=血色刀气，五雷=电弧 |
| 暴击 | 大字体 + 刀痕特效 |
| Boss出场 | 墨色渲染 + Boss名号 |
