## 二、P1 — Build多样性（下一里程碑）

| 系统 | 优先级 | 预估工期 | 依赖 |
|------|--------|---------|------|
| 传承(套装)机制 | **最高** | 5天 | P0装备系统 |
| 百炼坊(5大功能) | **最高** | 5天 | P0装备系统 |
| 武学秘录(3槽) | 高 | 2天 | 百炼坊萃取功能 |
| 传承UI | 高 | 2天 | 传承机制 |
| 百炼坊UI | 高 | 2天 | 百炼坊机制 |
| 集成测试 | **最高** | 2天 | 以上全部 |

**P1 核心价值**：让"Build构筑"从理论变为现实。玩家首次体验到"凑套装→伤害飞升"的爽感。

**P1 关键风险**：
- 套装数值平衡：6件效果的+400%是否会导致非套装Build完全不可用？→ 需要通过测试用例验证
- 百炼坊经济平衡：材料产出与消耗是否合理？→ 需要离线8小时测试场景验证

## 五、当前产品化缺口分析

### 已具备

- [x] 核心游戏循环（推图→掉落→换装→变强）
- [x] 7品质装备系统
- [x] 多乘区伤害公式
- [x] 离线挂机收益
- [x] GM调试工具

### P1急需

- [ ] 传承(套装) — 没有套装，Build千篇一律
- [ ] 百炼坊 — 没有材料消耗出口，锻材/精铁无限积累无意义
- [ ] 武学秘录 — 没有传奇特效激活栏，萃取无意义

### 产品体验缺口

- [ ] 战斗表演提升 — 当前战斗表演较简陋，需要更多视觉反馈
- [ ] 新手引导 — 无引导，新玩家不知道该做什么
- [ ] 音效/BGM — 目前无音效
- [ ] Steam商店页面准备 — 截图、描述、标签

## 六、里程碑验收标准

### P1 验收标准

1. 玩家可以凑齐一套6件传承并体验到DPS×5的飞升
2. 百炼坊5个功能全部可正常操作
3. 武学秘录3个槽位可正确激活传奇特效
4. `07_配置示例与测试用例.md` 中的全部测试用例通过
5. 10分钟游玩体验无阻塞性Bug

### P2 验收标准

1. 秘境可推到50层以上，数值缩放合理
2. 传奇宝石可正常掉落/升级/生效
3. 华戒可正确模拟2件套装效果
4. 第三章可完整游玩

### P3 验收标准

1. 宗师修为经验曲线合理（无尽但有意义）
2. 重入江湖后永久加成正确叠加
3. 二周目体验明显优于首轮（永久加成体感明显）
