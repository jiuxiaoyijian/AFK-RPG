### 2.2 体验循环（L2）

> 把 L1 的体验在六层时间尺度上分配兑现节奏。秒 / 分 / 时 / 日是底线，周 / 月可后续补。

| 时间尺度 | 玩家在这个尺度上获得什么 |
|---------|------------------------|
| 秒循环（< 3 秒） | （即时反馈：攻击命中 / 拾取 / 击杀） |
| 分循环（1-3 分钟） | （每 1-3 分钟获得的新鲜事） |
| 时循环（30-60 分钟） | （换装 / 解锁 / 过章大事件） |
| 日循环（一次登录周期） | （5 分钟能做什么、20 分钟能做什么） |
| 周循环（7 天） | （一周的成长里程碑） |
| 月循环（30 天+） | （终局目标 / 赛季 / 深渊） |
