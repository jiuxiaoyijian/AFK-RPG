# 项目进展记录 v3.16

## 1. 本轮定位

本轮继续按排期表推进，正式落地 `P0-2 Build 目标层` 首版。

目标不是再加一个复杂面板，而是把“当前 build 更缺什么、下一件该追什么、为什么刷这个节点”直接前置到主 HUD 和离线返回链路里。

## 2. 本轮目标

目标：

- 做三个核心流派的首版成长建议
- 给出当前 build 缺口诊断
- 给出下一件目标装备 / 词条提示
- 在 HUD 上直接展示推荐刷图理由
- 让每日刷装支线也能围绕当前缺口生成

## 3. 本轮完成内容

### 3.1 新增首版 build 诊断逻辑

本轮在 `GameManager` 中新增了统一的 build 建议输出。

当前已覆盖三条核心流派：

- `旋风斩`
- `深切创口`
- `连锁雷击`

每条流派现在都有：

- 当前阶段说明
- 新手成长摘要
- 核心优先属性组
- 缺口诊断逻辑
- 目标词条 / 传奇 / 底材推荐
- 推荐刷图节点与简短原因

这意味着 build 建议第一次从“玩家自己翻系统理解”变成了“系统直接告诉你下一步追什么”。

### 3.2 HUD 的 TargetCard 改成 build 目标卡

本轮没有新增更多卡片，而是把现有 `TargetCard` 直接升级为更偏 build 目标感的信息块。

它现在会展示：

- 当前追踪目标
- 当前 build 缺口
- 下一件优先目标
- 推荐刷图节点与简短理由

这样用户不打开图鉴和背包，也能直接知道：

- 当前是在追哪个传奇
- 现在更缺伤害、生存还是功能词条
- 下一件该找什么
- 为什么系统推荐去刷这个节点

### 3.3 每日刷装支线开始围绕 build 缺口生成

本轮同步改了 `DailyGoalSystem` 的刷装支线生成逻辑。

现在这条支线不再只是泛泛写“继续刷装”，而会优先围绕：

- 当前 build 的主要缺口
- 当前建议优先目标件
- 当前推荐刷图节点

这让 `P0-1 回流目标层` 和 `P0-2 Build 目标层` 开始真正串起来了。

### 3.4 离线收益弹窗补入 build 建议

本轮离线收益弹窗的“回来先做什么”分区也补了 build 建议内容。

现在离线回来后除了看到：

- 今日主目标
- 最近完成进度

还会直接看到：

- 当前构筑缺口
- 下一件优先目标

这进一步增强了回流后的方向感。
