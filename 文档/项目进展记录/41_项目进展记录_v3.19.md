# 项目进展记录 v3.19

## 1. 本轮定位

本轮进入 `P1-1 题材包装接入层` 的第一步，先完成文档层的题材切换与命名基线整理。

这轮不直接改游戏内文本，而是先把会影响后续实现的世界观、美术方向、命名表和协作文档全部统一，避免接下来边改边打架。

## 2. 本轮目标

目标：

- 正式放弃 `灾后黑铁废土` 方向
- 把项目题材切换到 `反桃花源修真`
- 产出一份可直接用于接入的命名包装表
- 同步修正美术需求、排期与协作记忆文件

## 3. 本轮完成内容

### 3.1 题材方向正式切换

本轮把项目的包装方向正式从：

- 灾后黑铁废土
- 工业遗迹
- 熔核残骸

切换为：

- 武侠 / 修仙
- 反桃花源
- 伪仙境
- 香火、祠庙与洞天秩序

新的包装核心已经明确为：

- 表面像桃源
- 内里是吃人的伪仙境
- 玩家不是来单纯飞升，而是来求生、破局、夺机缘

### 3.2 新增命名包装表

本轮新增了 `13_反桃花源修真命名包装表.md`。

它已经覆盖：

- 项目一句话卖点
- 玩家身份
- 三流派包装名
- 系统与 UI 命名映射
- 资源与货币命名建议
- 章节命名表
- 敌人与 Boss 命名映射
- 结果卡事件文案示例

这意味着后续替换章节名、面板标题、结果卡文本时，已经有了一份统一参考基线。

### 3.3 美术外包简报已同步到新题材

本轮把 `11_首批美术资源清单与外包需求表.md` 一并同步到了新方向。

现在它不再要求：

- 废土
- 工业遗迹
- 熔核科技

而改成了：

- 山水伪仙境
- 祠庙与灵田
- 香火秩序
- 越好看越不对劲

同时，首批章节场景、Boss、UI 和特效关键词也已经切到：

- `桃溪外渡`
- `祠下灵田`
- `伏祠山君`
- `祭炉监院`
- `御风 / 血劫 / 五雷`

### 3.4 排期和协作入口已同步

本轮还同步更新了：

- `12_开发排期表.md`
- `99_协作记忆文件.md`

这样下一轮再继续做游戏内接入时，不会再被旧题材方向误导。
