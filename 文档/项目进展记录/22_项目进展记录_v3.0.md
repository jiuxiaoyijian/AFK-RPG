# 项目进展记录 v3.0

## 1. 本轮定位

本轮承接前面多个版本逐步堆叠出来的原型，正式开始执行 `UI交互重构计划`，目标是把整体体验从“文字日志式原型”推进到更接近正式挂机产品的界面层。

本轮重点不再是补单个功能，而是重构：

- 信息层级
- 常驻 HUD
- 面板入口
- 面板交互秩序
- 反馈表现

## 2. 本轮目标

目标：

- 降低主界面的文字堆叠感
- 建立挂机游戏常见的 `常驻层 / 导航层 / 深度面板层`
- 统一背包、研究、图鉴、统计的入口和关闭逻辑
- 提升掉落、装备、追踪目标等高频信息的可读性

## 3. 本轮完成内容

### 3.1 HUD 从单块文字面板改为分区结构

原来的 HUD 把资源、节点、生命、流派、快捷键、目标、图鉴、装备、掉落全部堆在同一个 Panel 中，整体观感非常接近调试信息板。

本轮已拆分为多个区块：

- 顶部资源条
- 中部战斗信息卡
- 右侧目标追踪卡
- 左下装备概览卡
- 右下掉落反馈卡

这样玩家进入游戏后，能更清楚地一眼区分：

- 现在资源有多少
- 当前在打什么
- 当前流派和掉落方向是什么
- 当前追踪目标是什么
- 最近出了什么高价值掉落
- 现在身上四件装备大致如何

### 3.2 装备摘要改为槽位卡片

此前装备信息是多行文本摘要。

本轮已经把主 HUD 上的装备摘要改成四个槽位卡片：

- 武器
- 头盔
- 护甲
- 手套

每个槽位会显示：

- 装备名
- 装备评分
- 稀有度颜色

这样玩家不打开背包，也能快速判断当前 build 装备状态。

### 3.3 掉落摘要改为分层反馈

此前 HUD 中的掉落区更像一块结算日志。

本轮将其拆成：

- `高价值掉落`
- `最近掉落摘要`

高价值掉落会优先突出：

- 传奇相关掉落
- 装备类结果
- 自动分解结果

并使用更明显的颜色区分。

这样掉落反馈开始更接近挂机刷装游戏，而不是纯文本流水账。

### 3.4 新增底部统一导航栏

本轮新增了一个统一的主功能导航栏，取代“只能记快捷键”的入口方式。

导航栏当前承接：

- 背包
- 研究
- 图鉴
- 统计

每个按钮会显示简要摘要，例如：

- 背包物品数
- 可升级研究数量
- 当前追踪目标
- 当前统计样本概览

这样主界面已经具备更像正式产品的“功能导航层”。

### 3.5 新增统一面板管理器

此前所有二级面板互相独立，谁都能打开，谁也不会自动关，结果就是：

- 多层覆盖
- 快捷键误触
- 关闭逻辑混乱

本轮新增 `ui_overlay_manager.gd` 统一管理：

- 背包
- 研究
- 图鉴
- 掉落统计
- 离线报告

统一后实现了：

- 面板互斥显示
- `Esc` 关闭当前顶层面板
- 关闭按钮统一走同一套逻辑
- 离线报告也纳入同样的关闭流

### 3.6 面板打开时屏蔽战斗热键

此前在面板打开时，玩家误按 `1/2/3/R` 仍会触发战斗逻辑，例如：

- 切流派
- 重开节点

这对交互体验影响很大。

本轮通过 `EventBus.ui_blocking_input` 和统一 UI 状态管理，已实现：

- 只要有顶层面板打开，战斗热键输入就会被屏蔽

这样玩家在看背包、研究、图鉴、统计时，不会再因为误触而打断当前操作。

### 3.7 二级面板布局统一化

本轮同步把几个主要面板的布局统一成更一致的弹层风格：

- 统一更接近居中的大弹层
- 标题与关闭按钮位置统一
- 左侧列表 / 右侧详情的结构更一致
- 修复了原来研究面板一类的布局拥挤问题

虽然这还不是最终美术稿，但已经明显脱离“工具窗拼接感”。

### 3.8 视觉反馈层做了第一轮强化

本轮没有直接上复杂动画系统，而是先用低风险手段把关键状态做得更易读：

- 战斗状态文字按状态染色
- 流派名按流派颜色区分
- 装备槽位按稀有度颜色区分
- 目标卡与掉落卡用更明确的颜色强调重点

这属于第一轮“轻视觉化”，为后续继续做更完整的视觉反馈打了底。

## 7. 当前项目状态

截至 `v3.0`，原型的交互层已经从：

- 文字堆叠式 HUD
- 多面板自由重叠
- 只靠快捷键记忆

升级为：

- 分区式主 HUD
- 底部统一导航栏
- 统一 Overlay 面板管理
- Esc 关闭与输入阻塞
- 槽位化装备概览
- 更接近刷装游戏的掉落反馈

整体观感已经明显从“文字 MUD 原型”向“挂机刷装原型”迈出了一大步。

## 8. 当前已知问题

- 当前视觉强化主要依靠布局和颜色，还没有完整的图标和动画体系
- 二级面板虽然结构统一了，但还可以继续做更强的标签化和卡片化
- 统计面板的偏差结果仍然偏文本，后续适合改成条形/色块表达
- 顶部与底部功能层已经建立，但中心战斗展示区本身还可以继续增强演出感

## 9. 推荐下一步

按当前节奏，下一轮建议优先做：

- 给资源、装备、面板入口补图标
- 把统计面板的理论/实测偏差改成更直观的色块或进度条
- 强化掉落结算、离线奖励和传奇出现时的视觉冲击
- 继续提升战斗区的表现力，让主界面不仅“更清楚”，也“更像游戏”
