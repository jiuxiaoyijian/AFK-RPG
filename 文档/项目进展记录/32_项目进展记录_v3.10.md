# 项目进展记录 v3.10

## 1. 本轮定位

本轮继续沿上一轮的 HUD 收敛与战斗区保护方向推进，重点修正两个实际观感问题：

- 战斗安全框位置不符合真实战斗区域
- 二级页面打开时仍能看到收束动画

用户同时还要求：

- 装备概览与最近掉落继续上移

因此本轮的重点是把 `战斗舞台参考区` 调回正确位置，并进一步收敛战斗外 UI 对画面的干扰。

## 2. 本轮目标

目标：

- 让战斗安全框真正落在下半战斗区
- 让底部辅助卡和安全框位置关系更合理
- 打开二级页面时彻底隐藏收束动画

## 3. 本轮完成内容

### 3.1 战斗安全框下移到下半战斗区

上一轮新增的战斗安全框偏上，更像覆盖了中上区域，不符合实际主角、怪物、掉落主要发生的位置。

本轮已按用户截图意图重新调整：

- 战斗安全框横向几乎铺满主战斗区
- 纵向下移到更贴近角色、掉落和底部辅助卡所在区域

这样它现在更像：

- 真正的战斗舞台参考框
- 后续 HUD 自动避让时的可信边界

### 3.2 装备概览和最近掉落重新上移

在上一轮为了给安全框让位，底部两块辅助卡有些过低。

本轮重新做了平衡：

- `装备概览` 上移
- `最近掉落` 上移

这样它们既不会压进主战斗中心，又不会过度贴近底部导航栏。

### 3.3 二级页面打开时彻底隐藏收束动画

上一轮虽然已经把飞 UI 粒子层级降到二级页面之下，但在某些情况下仍然可能被感知到。

本轮进一步收紧规则：

- 只要任意二级页面打开
- `CollectEffects` 直接隐藏

这意味着现在打开：

- 背包
- 研究
- 图鉴
- 掉落统计
- GM 面板

时，收束动画都不会继续出现在画面里，避免影响页面阅读。
