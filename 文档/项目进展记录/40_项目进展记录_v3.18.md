# 项目进展记录 v3.18

## 1. 本轮定位

本轮继续按排期表推进，正式落地 `P0-4 观赏性增强层` 首版。

目标不是大改战斗系统，而是把挂机时最值得抬眼去看的时刻做得更容易被看到，包括：

- 精英 / Boss 遭遇
- 核心技能释放
- 传奇与追踪目标掉落

## 2. 本轮目标

目标：

- 增强精英 / Boss 遭遇提示
- 增强精英 / Boss 击破反馈
- 增强核心技能释放高光
- 增强传奇掉落与追踪目标掉落提示
- 在不大改结构的前提下提升挂机观看可读性

## 3. 本轮完成内容

### 3.1 HUD 新增战斗高光条

本轮在 HUD 上新增了一条统一的战斗高光区域。

它现在会在以下时机出现：

- 精英登场
- Boss 降临
- 精英击破
- Boss 击破
- 核心技能释放

这让玩家即使不盯着角色模型，也能更快捕捉到当前战斗是不是进入了关键节点。

### 3.2 精英 / Boss 遭遇与击破开始前台化

本轮把 `CombatSystem` 的波次与击杀链路补上了更明确的高光事件。

现在：

- 刷到精英时会出现显式提示
- 进入 Boss 波次时会出现显式提示
- 击杀精英 / Boss 时会有专门的击破高光

相比之前只靠状态文本变化，现在关键波次的辨识度明显更强了。

### 3.3 核心技能释放补入更明确的高光文案

本轮在玩家角色脚本里，为三条核心流派都补了释放时的高光提示：

- `旋风横扫`
- `裂伤爆发`
- `连锁雷击`

而且会附带简短副标题和观察提示，不再只是一个很轻的施法字样。

这让技能爆发瞬间在挂机观看中更容易形成节奏点。

### 3.4 伤害数字与掉落提示进一步强化

本轮还做了两类强化：

- 精英 / Boss 的受击数字反馈更有强调感
- 追踪目标与传奇掉落的 toast 更醒目、更持久

其中追踪目标掉落现在会：

- 用更明显的标题展示
- 使用更强的颜色与停留时间
- 在掉落视觉上使用更强的 beam / 尺寸 / 落地表现

这让“终于掉到了我要的东西”这件事更容易被玩家感知到。
