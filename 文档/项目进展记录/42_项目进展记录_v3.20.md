# 项目进展记录 v3.20

## 1. 本轮定位

本轮继续推进 `P1-1 题材包装接入层`，重点把命名包装表正式接到游戏内高频文本。

目标不是一次性改完所有描述和资源，而是先让玩家打开游戏后最常见的标题、章节名、敌人名、结果卡和阶段事件，先统一到 `反桃花源修真` 口径。

## 2. 本轮目标

目标：

- 把章节名接到游戏内
- 把敌人 / Boss 名接到游戏内
- 把三流派包装名接到游戏内

- 做一次 Godot 启动验证，确认文本接入没有引入解析错误

本轮直接修改了章节与敌人配置数据。

现在游戏内已切换为：

- `废墟边境` -> `桃溪外渡`
- `焦灼矿场` -> `祠下灵田`
- `腐鼠` -> `桃瘴鼠`
- `骨甲卫兵` -> `纸甲巡祠`
- `疫病萨满` -> `香灰巫祝`
- `铁爪巨兽` -> `伏祠山君`
- `灰烬掠夺者` -> `盗火佃客`
- `爆燃猎犬` -> `炽骨犬`
- `采场守卫` -> `守田傀`
- `锻炉圣卫` -> `护炉傩兵`
- `电链驯导者` -> `引雷役修`
- `熔核监工` -> `祭炉监院`

这意味着章节展示、敌人刷出、Boss 高光和结果卡读取到的显示名，已经开始统一。

### 3.2 三流派包装名已进入实际 UI

本轮把三个核心流派的显示名切换为：

- `御风道`
- `血劫道`
- `五雷道`

同时玩家角色高光文案也已经跟着调整，例如：

- `御风回斩`
- `血劫断命`
- `五雷连引`

这样玩家在首页和战斗高光里看到的，不再是旧的系统原型词汇。

### 3.3 高优先级 UI 文案已切换

本轮已完成一批最常见界面的题材接入：

- HUD 资源栏切到 `香火钱 / 祠灰 / 灵核 / 真意残片`
- `今日目标` 切到 `今日机缘`
- `研究` 切到 `悟道`
- `图鉴` 切到 `异闻录`
- `掉落统计` 切到 `机缘推演`
- `离线收益报告` 切到 `闭关所得`
- 导航栏提示从“流派 / 目标”切到“道统 / 机缘”

同时，DailyGoal、离线弹窗、阶段事件结果卡和一部分掉落反馈也都同步换成了新的说法。

### 3.4 阶段事件和结算文案开始统一

本轮还把阶段事件的标题体系往命名表收拢了。

现在会出现的标题方向包括：

- `首诛`
- `探境已开`
- `悟道初成 / 悟道圆满`
- `异宝现世`
- `机缘已成`

这使得 Boss 首杀、章节解锁、悟道升级和追踪达成，不再像旧题材残留的独立字符串。
