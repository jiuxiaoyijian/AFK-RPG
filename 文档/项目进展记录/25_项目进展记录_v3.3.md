# 项目进展记录 v3.3

## 1. 本轮定位

本轮承接 `v3.2` 已完成的 `分区化二级页`，继续把页面从“结构清楚”推进到“重点清楚”。

上一轮已经解决了：

- 二级页不再直接压在主界面上
- `背包 / 研究 / 图鉴 / 统计` 具备了基础分区

但实际观感上，页面仍然偏像“排好版的工具窗口”，还不够像一个正式挂机游戏的功能面板。

因此本轮重点是：

- 给分区增加更明确的视觉重心
- 给当前可操作状态增加一眼可读的提示行
- 把底部留白真正转化成可用的状态区与操作区

## 2. 本轮目标

目标：

- 让玩家打开二级页后，先看到“当前能做什么”，再去读正文
- 让同一页面里的不同区块具备更明显的颜色层次和职责
- 减少“只靠长文本说明当前状态”的问题

## 3. 本轮完成内容

### 3.1 背包页补上库存概览与操作提示

`背包` 面板本轮不再只有列表与详情，还新增了两层更直接的状态信息：

- 顶部 `库存概览`
- 底部 `操作提示`

其中顶部概览会显示：

- 当前库存总数
- 锁定数量
- 已装备槽位数
- 当前刷图掉落关注方向

底部操作提示会根据选中物品动态显示：

- 当前选中的装备名
- 当前是可装备 / 可分解 / 已锁定
- 当前评分

同时操作按钮改成更像“指令条”的三按钮并排布局：

- 装备到对应槽位
- 锁定 / 解锁
- 分解为铁屑

### 3.2 研究页补上升级状态提示

`研究` 面板本轮继续强化“能不能升、为什么不能升”的反馈：

- 顶部摘要改成两行概览
- 显示当前筛选分支的整体进度
- 显示金币 / 铁屑 / 核心等长期资源加成

底部 `成长操作` 区新增状态提示行，会直接告诉玩家：

- 当前未选择研究
- 当前研究可提升
- 下一跳消耗什么资源
- 或当前不可升级的原因

同时不同筛选按钮使用不同颜色，帮助玩家更快识别：

- 战斗分支
- 挂机分支
- 经济分支

### 3.3 图鉴页补上追踪状态提示

`图鉴` 面板本轮把“追踪”从一个按钮动作，提升成了明确的状态层：

- 顶部摘要同时展示图鉴完成度和当前追踪目标
- 底部 `追踪操作` 区新增状态提示行

状态提示会根据当前条目动态给出：

- 已发现 / 未发现
- 是否允许追踪
- 是否已经是当前追踪目标

同时：

- `传奇` 分类按钮做了更强的强调色
- 当前追踪目标按钮使用更高亮的金色状态
- 详情区文字颜色也会根据 `底材 / 词条 / 传奇` 分类切换

### 3.4 统计页补上快捷定位提示与更可读的最近样本

`统计` 面板本轮进一步压缩了“先看正文才能理解状态”的问题：

- 顶部摘要改成两行概览
- `快捷定位` 区新增提示行
- 会直接告诉玩家当前节点和推荐节点

同时：

- `当前节点` 与 `追踪推荐节点` 按钮做了不同颜色区分
- 最近样本从原来的长文本堆叠，改成更短的序号式记录
- 最近样本展示数量收敛到更适合当前区域高度的规模

这样统计页的视线顺序变成：

- 先看总样本与追踪目标
- 再看当前/推荐刷图入口
- 再看节点详情与效率条
- 最后看最近样本

## 4. 本轮关键实现点

本轮主要是 `场景布局扩展 + 控制器状态提示 + 轻量配色强化`：

- `inventory_panel.tscn`：新增库存概览和操作提示，并把操作按钮整理为一排
- `research_panel.tscn`：扩展成长操作区，加入研究状态提示
- `codex_panel.tscn`：扩展追踪操作区，加入追踪状态提示
- `drop_stats_panel.tscn`：扩展最近样本区，加入快捷定位提示
- `inventory_panel_controller.gd`：增加库存概览构建、物品状态提示和稀有度配色
- `research_panel_controller.gd`：增加研究摘要构建、升级状态提示和分支配色
- `codex_panel_controller.gd`：增加图鉴摘要构建、追踪状态提示和分类配色
- `drop_stats_panel_controller.gd`：增加统计摘要构建、快捷定位提示和近期样本压缩展示

## 7. 当前项目状态

截至 `v3.3`，二级页已经从：

- 只是功能分栏

继续推进到：

- 有分区
- 有状态提示
- 有颜色重点
- 有更接近正式游戏面板的阅读顺序

也就是说，现在这些页面已经不只是“能打开”，而是开始具备：

- 先看状态
- 再看细节
- 最后执行操作

这样的典型挂机游戏 UI 节奏。

## 8. 当前已知问题

- 仍然缺少真正的图标、美术框体和按钮资产
- `ItemList` 仍是通用控件，条目层级还不够像正式卡片列表
- 主界面战斗区虽然已经让出空间，但战斗表现本身还不够有“挂机观赏性”

## 9. 推荐下一步

按当前节奏，下一轮建议优先做：

- 给资源、分支、追踪、稀有度补轻量伪图标或正式图标槽位
- 把 `最近掉落 / 最近样本 / 离线收益` 统一做成结果卡片风格
- 继续加强首页战斗区动效和信息锚点，让主界面更接近真正可观看的挂机战斗
