> 基于 `14_暗黑3式系统框架设计.md` 第七至第十一节。

## 一、传承系统（P1）

### 1.1 设计理念

传承（套装）= 暗黑3套装。提供巨额乘区E加成，是Build定义性的核心系统。

- **每套传承6件**，触发效果分3档（2件/4件/6件）
- **6件效果极强**（+300%~+500%），定义Build方向
- **玩家同时只能凑1套传承**（装备9件，传承件最多占6件）
- **华戒类饰品**可让非传承件获得某套传承的2件效果，支持混搭

### 1.2 四大传承详细设计

#### 御风传承 (`set_wind`)

| 件数 | 效果 | 数值 |
|------|------|------|
| 2件 | 御风刀范围扩大 | 范围 +30% |
| 4件 | 御风刀命中敌人后获得"风势"层，每层提升攻速 | 每层 +5% 攻速，最多10层 |
| 6件 | 御风刀造成的伤害提升 | **御风刀伤害 +400%** |

**Build方向**：持续旋风，依靠6件的巨额加成碾压，适合范围清场。

#### 血劫传承 (`set_blood`)

| 件数 | 效果 | 数值 |
|------|------|------|
| 2件 | 流血效果持续时间延长 | 流血持续 +3秒 |
| 4件 | 流血目标受到的所有伤害提升 | 流血目标 +80% 受伤 |
| 6件 | 血劫手和流血总伤害提升 | **血劫手 + 流血伤害 +350%** |

**Build方向**：叠流血→爆裂→处决，单体Boss最强。

#### 五雷传承 (`set_thunder`)

| 件数 | 效果 | 数值 |
|------|------|------|
| 2件 | 五雷掌连锁次数增加 | 连锁 +2次 |
| 4件 | 被雷击过的敌人受到的雷伤增加 | 雷击目标 +60% 受雷伤 |
| 6件 | 五雷掌伤害提升 | **五雷掌伤害 +350%** |

**Build方向**：连锁弹射覆盖大量敌人，均衡表现。

#### 铁壁传承 (`set_iron`)

| 件数 | 效果 | 数值 |
|------|------|------|
| 2件 | 格挡率提升 | 格挡率 +15% |
| 4件 | 格挡成功后反弹伤害 | 反弹 200% 格挡值的伤害 |
| 6件 | 受到伤害时积蓄怒意，满蓄后释放爆发攻击 | **每积蓄100%生命值伤害→释放AOE = 积蓄伤害 ×500%** |

**Build方向**：坦克反伤流，不依赖特定武技流派，任何流派均可搭配。

### 1.3 传承件分布

每套传承覆盖6个槽位，具体分布：

| 传承 | 兵器 | 护额 | 护甲 | 护手 | 腿甲 | 轻靴 | 腰封 | 佩饰 |
|------|------|------|------|------|------|------|------|------|
| 御风 | ✓ | ✓ | ✓ | ✓ | ✓ | ✓ | — | — |
| 血劫 | ✓ | ✓ | ✓ | ✓ | ✓ | — | ✓ | — |
| 五雷 | ✓ | ✓ | ✓ | ✓ | — | ✓ | — | ✓(左) |
| 铁壁 | — | ✓ | ✓ | ✓ | ✓ | ✓ | ✓ | — |

### 1.4 华戒（传承辅助饰品）

| 名称 | 效果 | 获取 |
|------|------|------|
| 御风华戒 | 佩戴后视为身穿2件御风传承 | 试剑秘境20层+ 掉落 |
| 血劫华戒 | 佩戴后视为身穿2件血劫传承 | 试剑秘境20层+ 掉落 |
| 五雷华戒 | 佩戴后视为身穿2件五雷传承 | 试剑秘境20层+ 掉落 |
| 铁壁华戒 | 佩戴后视为身穿2件铁壁传承 | 试剑秘境20层+ 掉落 |

**混搭示例**：穿6件御风传承(6件效果) + 1个血劫华戒(佩饰) = 同时获得御风6件 + 血劫2件效果。

## 二、百炼坊系统（P1）

### 2.1 设计理念

百炼坊 = 暗黑3卡奈魔盒。提供装备精炼、萃取、升级、转化等功能，是材料消耗的主要出口。

### 2.2 五大功能

#### 功能1: 萃取武学 (`extract`)

将一件真意或以上品质的装备分解，永久解锁其传奇特效到"武学秘录"中。

| 参数 | 值 |
|------|-----|
| 输入 | 1件真意+品质装备 |
| 消耗 | 50精铁 + 10秘卷残页 |
| 输出 | 永久解锁该传奇特效到武学秘录 |
| 限制 | 武学秘录最多激活3个传奇特效（兵器/护甲/饰品各1） |

#### 功能2: 精钢化真 (`upgrade_rare`)

将一件玄品装备升级为真意品质（随机传奇特效）。

| 参数 | 值 |
|------|-----|
| 输入 | 1件玄品装备 |
| 消耗 | 80锻材 + 30精铁 |
| 输出 | 1件真意品质装备（保留原有3条词缀，新增1条词缀+1条传奇特效） |
| 限制 | 无 |

#### 功能3: 回炉重铸 (`reforge`)

将一件真意装备的所有词缀完全重新随机。

| 参数 | 值 |
|------|-----|
| 输入 | 1件真意+品质装备 |
| 消耗 | 100锻材 + 20精铁 |
| 输出 | 同品质同槽位装备，词缀全部重随 |
| 限制 | 品质不变，套装标记不变 |

#### 功能4: 传承互转 (`convert_set`)

将一件传承装备转换为同套传承内的其他槽位。

| 参数 | 值 |
|------|-----|
| 输入 | 1件传承品质装备 |
| 消耗 | 30精铁 + 5秘卷残页 |
| 输出 | 同套传承的随机其他槽位装备（词缀重随） |
| 限制 | 只能在同一套传承内转换 |

#### 功能5: 淬火精炼 (`reroll`)

重新随机一件装备上的一条词缀。

| 参数 | 值 |
|------|-----|
| 输入 | 1件装备 + 选定1条词缀 |
| 消耗 | 50锻材（每次递增10%） |
| 输出 | 选定词缀被替换为同bucket内的另一条（或同词缀新数值） |
| 限制 | 每件装备只能精炼1条词缀（锁定精炼槽） |

### 2.3 材料来源与消耗平衡

| 材料 | 主要来源 | 消耗出口 |
|------|---------|---------|
| 碎银 | 所有掉落分解 | 基础操作费用 |
| 锻材 | 凡品/精良分解 | 精钢化真、回炉重铸、淬火精炼 |
| 精铁 | 玄品分解、每日修行 | 萃取、精钢化真、传承互转 |
| 秘卷残页 | 真意+分解、Boss击杀 | 萃取、传承互转 |

## 三、武学秘录系统（P1，百炼坊子系统）

### 3.1 设计

武学秘录 = 暗黑3卡奈魔盒的传奇特效栏。玩家通过萃取功能解锁传奇特效，然后在3个槽位中各选1个激活。

```
武学秘录:
  ┌─────────┐  ┌─────────┐  ┌─────────┐
  │ 兵器栏   │  │ 护甲栏   │  │ 饰品栏   │
  │ (选1激活) │  │ (选1激活) │  │ (选1激活) │
  └─────────┘  └─────────┘  └─────────┘

已解锁的传奇特效:
  风眼 ✓  血诏 ✓  雷暴矩阵 ○  铁壁心法 ✓  ...
```

### 3.2 规则

1. 萃取后永久解锁（不消耗源装备以外的东西）
2. 3个激活槽（兵器/护甲/饰品），各选1个传奇特效激活
3. 激活的传奇特效效果等同于装备上携带该传奇词缀
4. 可随时免费切换已解锁的特效
5. 武学秘录的效果叠加在装备本身的传奇特效之上

## 四、试剑秘境系统（P2）

### 4.1 设计理念

试剑秘境 = 暗黑3大秘境。无限推层的终局内容，是衡量Build强度的标尺。

### 4.2 核心机制

| 要素 | 设计 |
|------|------|
| 入场 | 消耗秘境钥石（Boss掉落） |
| 层数 | 1层起，无上限 |
| 目标 | 在限时内清光所有敌人 |
| 奖励 | 层数越高，掉落品质加成越高 |
| 排行 | 个人最高层记录 |

### 4.3 数值缩放表

| 层数 | 怪物血量倍率 | 怪物攻击倍率 | 掉落品质加成 | 真意掉率 |
|------|-------------|-------------|-------------|---------|
| 1-5 | 1.0~1.5x | 1.0~1.2x | +0% | 7% |
| 6-10 | 1.5~3.0x | 1.2~1.8x | +10% | 8% |
| 11-20 | 3.0~8.0x | 1.8~3.0x | +25% | 10% |
| 21-30 | 8.0~20x | 3.0~5.0x | +50% | 13% |
| 31-50 | 20~80x | 5.0~10x | +100% | 18% |
| 51+ | 80x × 1.17^(n-50) | 10x × 1.1^(n-50) | +150% | 22% |

每层增加 ≈ 17% 怪物血量，需要 ≈ 17% DPS 提升才能维持同样的通关速度。

### 4.4 秘境钥石

| 钥石等级 | 来源 | 对应层数范围 |
|---------|------|------------|
| 粗铁钥石 | 章节Boss掉落 | 1-10层 |
| 精钢钥石 | 秘境10层+掉落 | 11-30层 |
| 玄铁钥石 | 秘境30层+掉落 | 31-50层 |
| 陨铁钥石 | 秘境50层+掉落 | 51层+ |

### 4.5 秘境专属掉落

| 掉落 | 层数要求 | 说明 |
|------|---------|------|
| 华戒 | 20层+ | 传承辅助饰品 |
| 传奇宝石 | 10层+ | 佩饰专属宝石 |
| 宝石升级石 | 通关奖励 | 用于升级已有传奇宝石 |

## 五、传奇宝石系统（P2，秘境子系统）

### 5.1 设计

传奇宝石 = 暗黑3传奇宝石。镶嵌在佩饰槽位中，提供独特效果。

### 5.2 宝石列表

| 宝石名 | 效果 | 每级提升 | 最高等级 |
|--------|------|---------|---------|
| 猛虎之石 | 攻击力 +X% | +1% | 50 |
| 铁壁之石 | 防御力 +X% | +1% | 50 |
| 迅雷之石 | 攻速 +X% | +0.5% | 50 |
| 噬血之石 | 击杀回复 X% 生命 | +0.2% | 50 |
| 破军之石 | 暴击伤害 +X% | +2% | 50 |
| 困兽之石 | 精英伤害 +X% | +1.5% | 50 |

### 5.3 升级机制

```
宝石升级:
  1. 通关秘境获得"宝石升级石"
  2. 升级成功率: 100% (等级≤通关层数时), 60% (等级=层数+1~5), 30% (等级>层数+5)
  3. 升级失败不降级
```

## 六、宗师修为系统（P3）

### 6.1 设计理念

宗师修为 = 暗黑3巅峰等级。角色达到满级后，继续获得经验转化为宗师修为点，提供缓慢但无上限的属性增长。

### 6.2 修为面板

| 分类 | 可分配属性 | 每点收益 | 上限 |
|------|-----------|---------|------|
| 攻击 | 力量/攻击力% | +0.5% | 每属性50点 |
| 防御 | 生命值/防御力 | +0.5% | 每属性50点 |
| 技巧 | 暴击率/暴击伤害 | +0.2%/+1% | 每属性50点 |
| 辅助 | 碎银获取/经验获取 | +1% | 无上限 |

### 6.3 经验曲线

```
宗师修为等级 N 所需经验 = 基础经验 × (1 + 0.05 × N)^2

前20级: 约每2小时1级（正常游玩）
50级后: 约每6小时1级
100级后: 约每12小时1级（缓增，永不到顶）
```

## 七、重入江湖系统（P3）

### 7.1 设计理念

重入江湖 = 暗黑3赛季。玩家可选择"重入江湖"，放弃当前角色进度，获得永久加成，重新开始。

### 7.2 核心机制

| 要素 | 设计 |
|------|------|
| 触发条件 | 角色达到满级 + 秘境通关20层以上 |
| 保留 | 武学秘录（已萃取的传奇特效）、宗师修为点、江湖见闻录 |
| 重置 | 等级、装备、材料、章节进度 |
| 永久加成 | 每次重入获得"江湖阅历"被动加成 |

### 7.3 江湖阅历（永久被动）

| 重入次数 | 获得加成 |
|---------|---------|
| 第1次 | 全属性 +5%, 经验获取 +10% |
| 第2次 | 全属性 +10%, 经验获取 +20%, 掉落品质 +5% |
| 第3次 | 全属性 +15%, 经验获取 +30%, 掉落品质 +10%, 碎银获取 +15% |
| 第N次 | 递增(每次+5%属性, +10%经验, +5%掉落, +5%碎银) |
| 上限 | 无硬上限，但收益递减 |

### 7.4 赛季规划

每个"赛季"持续约3个月，赛季结束时：
1. 赛季角色合并到主角色
2. 赛季排行榜定格
3. 赛季专属奖励发放（赛季限定外观/称号）

## 十、验收标准

### P1验收

1. 四套传承各6件可正常掉落
2. 2/4/6件效果正确触发并体现在伤害计算中
3. 百炼坊5大功能均可正常使用
4. 武学秘录3槽激活正确生效
5. 材料消耗/产出循环合理

### P2验收

1. 秘境可正常推层，数值缩放正确
2. 传奇宝石可掉落、镶嵌、升级
3. 华戒可掉落并正确模拟2件套装效果

### P3验收

1. 宗师修为经验曲线正确
2. 重入江湖保留/重置项正确
3. 江湖阅历永久加成正确叠加
