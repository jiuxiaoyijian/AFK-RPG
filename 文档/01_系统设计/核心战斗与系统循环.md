# 核心战斗与系统循环设计

> 基于 `14_暗黑3式系统框架设计.md` 第三节与第十三节。
> 命名参照 `13_武侠命名映射表.md`。

---

## 一、核心玩法定义

横版2D自动战斗挂机RPG。玩家不直接操作战斗，但战斗**必须好看、必须可读**，让玩家能"看懂"自己的Build在起作用。

### 1.1 战斗角色分工

| 角色 | 说明 |
|------|------|
| 玩家角色 | 画面中央偏左，自动释放招式 |
| 普通敌人 | 左右两侧涌入，被清场 |
| 精英敌人 | 单独出场，有特殊标记 |
| Boss | 独占画面右侧，有血条和名号 |

### 1.2 战斗表演层

| 元素 | 设计 |
|------|------|
| 招式释放 | AI按优先级自动释放，屏幕显示招式名+特效 |
| 身法闪避 | 角色遇大招自动闪避(有概率) |
| 怪物密度 | 根据节点配置生成波次 |
| 战斗节奏 | 由攻速/招式CD决定，玩家可通过装备影响 |

## 二、三层循环

### 2.1 分钟循环（战斗内）

```
进入节点 → 怪物涌入 → 自动战斗 → 掉落拾取 → 下一波/通关
```

每个普通节点 5-10秒/波，精英 15-20秒，Boss 30-60秒。

### 2.2 章节循环（数小时）

```
选择章节节点 → 通关获取装备 → 比较装备/调整Build → 尝试下个节点
```

每章包含 3-4 个普通节点 + 1个精英 + 1个Boss。卡关时系统建议回退刷装备。

### 2.3 长线循环（数天-数周）

```
推图毕业 → 试剑秘境推层 → 宗师修为 → 重入江湖(可选) → 永久加成 → 再来一轮
```

## 三、战斗可读性反馈

| 触发事件 | 视觉反馈 |
|---------|---------|
| 传承特效触发 | 全屏闪光 + 文字提示 "御风传承·六件效果触发" |
| 暴击 | 数字放大 + 颜色变化(金色) |
| 传奇特效触发 | 专属动画 + 音效 |
| 濒死触发 | 屏幕边缘泛红 → 防御特效激活 |
| 秒杀 | 快速清屏表演 → 满足感 |
| 流血叠层 | 敌人身上出现血色标记，叠层数字 |
| 连锁雷击 | 电弧在多个敌人间弹射 |

---

## 四、战斗速度控制

| 节点类型 | 战斗时长 | 侧重 |
|---------|---------|------|
| 普通节点 | 5-10秒/波 | 效率和掉落数量 |
| 精英节点 | 15-20秒 | 更多特效触发，掉落品质更高 |
| Boss节点 | 30-60秒 | 全特效展示，可能掉落传承/真意 |
| 秘境层 | 根据层数动态调节 | 卡关时战斗明显变慢→提示需要更好装备 |

## 五、技能结构

### 5.1 技能分类

| 类型 | 说明 | 释放方式 |
|------|------|---------|
| 基础攻击 | 无CD无消耗的普通攻击 | 自动持续释放 |
| 核心武技 | 流派主技能，定义Build方向 | CD好即释放 |
| 战术武技 | 辅助技能，增幅/控制 | 条件触发 |
| 爆发武技 | 长CD高伤害，观赏性最强 | CD好即释放 |
| 被动心法 | 常驻增幅效果 | 被动生效 |

### 5.2 三大流派核心武技

#### 御风刀 (`core_whirlwind`)

| 参数 | 值 |
|------|-----|
| 名称 | 御风刀 |
| 类型 | 自身周围范围 |
| 武器限制 | 刀/斧 (`blade`, `axe`) |
| CD | 5.5秒 |
| 内力消耗 | 20 |
| 基础倍率 | 1.2x |
| 命中次数 | 6次（持续2.5秒，每0.4秒1次） |
| 成长 | 每级 +0.1 倍率 |
| 战斗表现 | 角色原地旋转，周围产生刀风，范围内敌人持续受伤 |

#### 血劫手 (`core_deep_wound`)

| 参数 | 值 |
|------|-----|
| 名称 | 血劫手 |
| 类型 | 单体近战 |
| 武器限制 | 刀/枪 (`blade`, `spear`) |
| CD | 4.8秒 |
| 内力消耗 | 18 |
| 基础倍率 | 1.6x |
| 命中次数 | 1次 |
| 成长 | 每级 +0.12 倍率 |
| 战斗表现 | 快速突刺，造成高额单体伤害并附带流血DOT |

#### 五雷掌 (`core_chain_lightning`)

| 参数 | 值 |
|------|-----|
| 名称 | 五雷掌 |
| 类型 | 远程连锁 |
| 武器限制 | 法球/杖 (`orb`, `staff`) |
| CD | 6.0秒 |
| 内力消耗 | 22 |
| 基础倍率 | 1.4x |
| 命中次数 | 1次（连锁另算） |
| 成长 | 每级 +0.1 倍率 |
| 战斗表现 | 掌心射出雷电，命中后连锁弹射至附近敌人 |

## 九、离线战斗规则

```
离线期间:
  - 正常计算战斗轮次和掉落
  - 凡品/精良 → 自动分解为锻材(上线时汇总显示)
  - 玄品 → 存入"待鉴定"队列(上线后手动查看)
  - 真意及以上 → 存入"出关所得·高光"(上线后逐个展示掉落动画)

上线时:
  "出关所得"弹窗 → 先显示总收益 → 再逐个播放高品质掉落的表演动画
```

## 十、战斗系统验收标准

1. 三个流派的核心武技均可正常释放并产生预期伤害
2. 战斗表演流畅，无明显卡顿
3. 掉落物品品质与节点/敌人类型匹配
4. 离线战斗产出与在线成比例
5. Boss战有明显的阶段感和视觉冲击力
6. 装备变更后，战斗表现有可感知的变化
