## 体验设计（L1-L5）

> 本章节依据 [体验分析框架](../00_项目总纲/体验分析框架.md) L1-L5 方法论填写，回应 [Phase C 审计](../06_会议与决策/2026-03-19_存量系统设计L1-L5审计.md) 指出的 P0 级体验锚点 / 秒循环音画时序 / 学习曲线空白。
>
> 本章节**先于**后续功能规格（第一节起），作为整个战斗系统的体验承诺。

### 体验锚点（L1）

#### 核心幻想（Fantasy）

玩家是一位初出青云门的年轻弟子，第一次独自下山历练。画面中央的那个小人是**自己运筹帷幄的化身** —— 不去指挥每一下出招，但每一次清场、每一次濒死反转、每一次传承触发，都是**"我的 Build 在起作用"的旁观者证据**。

> 一句话总结：**"这不是在操作战斗，是在见证自己的修行成形。"**

#### 核心情绪（Emotion）

- **主情绪 · 掌控感**：玩家关掉声音也能看懂每一次伤害跳跃、每一次濒死反转的来由
- **次情绪 · 期待**：下一波怪里有没有精英？这只 Boss 会不会掉到橙字？
- **反面警戒**：本系统**不以操作爽感为目标**（手动挂机不适合），所以"掌控感"必须通过**可读性**而不是输入反馈来兑现

#### 核心爽点（Payoff）

一场代表性战斗的高光序列（可反复经历，可截图片段化）：

1. 精英节点半程，玩家已清掉 2 波喽啰
2. 第 3 波涌入时，传承 6 件套满足触发条件 →  全屏金色闪光 + "御风传承·六件效果触发"飞字
3. 角色原地旋转，**3 秒内清光整场喽啰波**（集群清场节奏）
4. 精英独自上场 → **≤ 1 秒秒杀**，金色光柱从尸体升起（单体秒杀节奏）
5. 拾取慢镜特写 → 详情弹出 **"绝世真意"**
6. 背包 DPS 数字栏从 8,500 跳到 12,200
7. 右下角 "出关所得" 计数 +1

> **口径统一**：本项目区分两种高光节奏 —— **集群清场 ≈ 3 秒**（御风刀式），**单体秒杀 ≤ 1 秒**（血劫手式）。两者共同构成"Build 在起作用"的可读性证据，与 L5 仪式清单（#6 秒杀精英 = 1 秒内清屏）一致。

### 体验循环（L2）

#### 秒循环 · 音画时序轴

本表是对第三节"战斗可读性反馈"表的**音画时序化补强**。每类事件在 0-100ms / 100-300ms / 300-800ms / 800ms+ 四个时间窗内的视听设计：

| 事件 | 0-100 ms | 100-300 ms | 300-800 ms | 800 ms+ |
|------|---------|-----------|-----------|---------|
| 传承 6 件触发 | 屏幕金色闪光 + 低频号角音 | "御风传承·六件效果触发" 飞字 | 光晕渐隐 + BGM 短暂拔高 | 战斗延续，角色 aura 常驻 2 秒 |
| 暴击 | 数字从 1 倍放大到 1.5 倍 + 金色描边 | 粒子沿数字方向扩散 | 数字弧线下落 | 下一次暴击可叠加特效闪频 |
| 传奇特效触发 | 专属特效动画 + 专属音效 | 传奇图标在 HUD 闪光 | 效果持续期间角色脚下有环形提示 | 效果结束回归静音 |
| 濒死触发 | 屏幕边缘 0.3 秒内泛红 | 防御特效激活音 + 屏幕震动 | 泛红缓慢收敛 | 角色血条上方出现"铁壁"标记 |
| 秒杀 | 快速清屏特效 + 高频打击音连续 | 敌人倒地弹跳 | 残影消散 + 掉落弹出 | — |
| 流血叠层 | 敌人身上血色数字+1 | 血色印记持续闪烁 | 叠层数字角标跳动 | 满层时印记变金色预警即将爆裂 |
| 连锁雷击 | 电弧音 + 屏幕偏蓝 | 电弧在敌人间弹射（每跳 60ms） | 电弧收束 + 最后一击闪光 | — |

**判定标准**：关闭声音，光看画面能判断出至少 5 类事件的类型。

#### 分循环（1-3 分钟）

| 节点类型 | 时长 | 期间发生的"新鲜事" |
|---------|------|------------------|
| 普通节点（3 波） | 90-120 秒 | 每波 5-10 秒 + 波间 2 秒换波动画 + 波次结算数字飞行 |
| 精英节点 | 120-150 秒 | 精英出场特写 + 被 DOT 标记的叠层视觉 + 击杀后 100% 玄品以上掉落 |
| Boss 节点 | 180-240 秒 | Boss 血条分段 + 阶段切换音效 + 通关后高品质掉落表演 |

#### 时循环（30-60 分钟）

- **通关新章节 Boss**：解锁下一章场景（视差背景换章节）
- **每小时保底**：≥ 1 件玄品 + ≥ 0.3 概率 1 件真意
- **战术分叉点**：每小时末会有一次"继续推图 vs 回刷本章凑套装"的决策窗口

#### 日循环

| 时段 | 玩家能做什么 |
|------|-------------|
| 5 分钟 | 收取闭关收益 → 完成今日机缘 1-2 条（无脑领取节奏） |
| 20 分钟 | 推 2-3 个章节节点 + 试一次装备替换 + 整理背包 |
| 4-8 小时（离线） | 自动战斗积累，上线时看 "出关所得" 高光回放 |

**判定**：只玩 5 分钟的玩家，明天仍愿意再登录的比例 ≥ 60%。

#### 周循环

- 第 3 天：凑齐第一套传承 2 件套，4 件套效果触发
- 第 5 天：宗师修为系统解锁
- 第 7 天：秘境首次破 10 层 / 解锁精钢钥石

#### 月循环

- 第 14 天：凑齐一套 6 件传承（完整 Build，与[装备·数值·Build系统 · L2 月循环](装备·数值·Build系统.md#月循环) 口径一致）
- 第 30 天：宗师修为 50 级
- 第 90 天（赛季）：重入江湖，获得江湖阅历 +1

### 决策空间（L3）

三流派差异化（第五节）+ 战术 / 爆发 / 被动武技四层结构为决策骨架。**量化补强项**（见修订优先级 P1）：

- 战术 / 爆发 / 被动武技的实例集待补（每类 ≥ 3 个）
- 流派切换代价：传奇武学萃取后跨流派通用，但传承件必须重新凑

### 学习曲线（L4）

| 里程碑 | 玩家在战斗中应当理解 / 解锁 |
|--------|---------------------------|
| **5 分钟** | 看见自动战斗在释放 + 看见经验条在涨 + 看见怪物尸体飞出物品（"打怪 = 得东西"的闭环） |
| **1 小时** | 打通第一章 Boss（铁面虬髯客）+ 第一次看到橙字传奇特效的专属动效 + HUD "当前 Build" 卡首次给出建议 |
| **10 小时** | 主动选定流派（御风 / 血劫 / 五雷）+ 开始凑 2-4 件同套传承 + 第一次使用百炼坊萃取 |
| **100 小时** | 秘境 30 层+ / 宗师修为 50+ / 完成第一次重入江湖 / 至少对比过 2 个流派的 Build |

**难度曲线校验**：无 5 分钟 → 1 小时的认知断档；1 小时 → 10 小时之间由"百炼坊解锁"作为主要跨越动作承接。

### 情感反馈（L5）

#### 仪式感清单（在原第三节基础上扩展）

1. **传承 6 件触发**（见秒循环时序轴）
2. **橙字掉落**：尸体金光柱（1.5 秒）→ 拾取慢镜（0.4 秒）→ 详情弹出（"绝世真意"字体特殊）→ DPS 跃升数字对比
3. **Boss 斩杀**：画面短暂定格 0.3 秒 → 血溅横扫 → "击破 · 铁面虬髯客" 文字 → 掉落高光序列
4. **出关所得**：离线上线弹窗 → 先显示总收益 → 逐个播放真意+品质掉落动画（第九节已定义）
5. **濒死反转**：屏幕泛红 → 防御 / 铁壁触发 → 玩家从视觉上感受到"Build 在救我"
6. **秒杀精英**：1 秒内清屏，伤害数字堆叠爆量

#### 挫败缓冲（待补 · P2 级）

- 卡关时的"软着陆"：系统建议（第二节已提及"回退刷装备"）需要**有声无声**的表达差（是否触发建议弹窗 vs 打断玩家）
- 离线战斗打不过节点：自动回退至可通过的前一节点（不在本文档职责内，属于离线系统）

#### 可分享锚点（零 / 极低工程成本取舍）

- **秒杀 Boss 可分享**：**不内置录屏**。预留战斗摘要数据接口（Boss 名 / 战斗时长 / 最大单次伤害数字 / 使用流派），让玩家使用**系统级截图**（Win+Print Screen / Steam 截图 / 手机截屏）或外部录屏工具即可
- **绝世真意掉落可分享**：**不做内置截图模式**。依靠详情面板本身的**高对比度 + 干净信息层级**，让玩家系统级截图就能出片
- **设计约束**：可分享性的工程成本**刻意压到 0 或极低**，美术 / 工程不因此多出新功能；UI 层面的干净与可读性才是"可分享"的唯一保障

## 一、核心玩法定义

横版2D自动战斗挂机RPG。玩家不直接操作战斗，但战斗**必须好看、必须可读**，让玩家能"看懂"自己的Build在起作用。

### 1.1 战斗角色分工

| 角色 | 说明 |
|------|------|
| 玩家角色 | 画面中央偏左，自动释放招式 |
| 普通敌人 | 左右两侧涌入，被清场 |
| 精英敌人 | 单独出场，有特殊标记 |
| Boss | 独占画面右侧，有血条和名号 |

### 1.2 战斗表演层

| 元素 | 设计 |
|------|------|
| 招式释放 | AI按优先级自动释放，屏幕显示招式名+特效 |
| 身法闪避 | 角色遇大招自动闪避(有概率) |
| 怪物密度 | 根据节点配置生成波次 |
| 战斗节奏 | 由攻速/招式CD决定，玩家可通过装备影响 |

## 二、三层循环

### 2.1 分钟循环（战斗内）

```
进入节点 → 怪物涌入 → 自动战斗 → 掉落拾取 → 下一波/通关
```

每个普通节点 5-10秒/波，精英 15-20秒，Boss 30-60秒。

### 2.2 章节循环（数小时）

```
选择章节节点 → 通关获取装备 → 比较装备/调整Build → 尝试下个节点
```

每章包含 3-4 个普通节点 + 1个精英 + 1个Boss。卡关时系统建议回退刷装备。

### 2.3 长线循环（数天-数周）

```
推图毕业 → 试剑秘境推层 → 宗师修为 → 重入江湖(可选) → 永久加成 → 再来一轮
```

## 四、战斗速度控制

| 节点类型 | 战斗时长 | 侧重 |
|---------|---------|------|
| 普通节点 | 5-10秒/波 | 效率和掉落数量 |
| 精英节点 | 15-20秒 | 更多特效触发，掉落品质更高 |
| Boss节点 | 30-60秒 | 全特效展示，可能掉落传承/真意 |
| 秘境层 | 根据层数动态调节 | 卡关时战斗明显变慢→提示需要更好装备 |

## 五、技能结构

### 5.1 技能分类

| 类型 | 说明 | 释放方式 |
|------|------|---------|
| 基础攻击 | 无CD无消耗的普通攻击 | 自动持续释放 |
| 核心武技 | 流派主技能，定义Build方向 | CD好即释放 |
| 战术武技 | 辅助技能，增幅/控制 | 条件触发 |
| 爆发武技 | 长CD高伤害，观赏性最强 | CD好即释放 |
| 被动心法 | 常驻增幅效果 | 被动生效 |

### 5.2 三大流派核心武技

#### 御风刀 (`core_whirlwind`)

| 参数 | 值 |
|------|-----|
| 名称 | 御风刀 |
| 类型 | 自身周围范围 |
| 武器限制 | 刀/斧 (`blade`, `axe`) |
| CD | 5.5秒 |
| 内力消耗 | 20 |
| 基础倍率 | 1.2x |
| 命中次数 | 6次（持续2.5秒，每0.4秒1次） |
| 成长 | 每级 +0.1 倍率 |
| 战斗表现 | 角色原地旋转，周围产生刀风，范围内敌人持续受伤 |

#### 血劫手 (`core_deep_wound`)

| 参数 | 值 |
|------|-----|
| 名称 | 血劫手 |
| 类型 | 单体近战 |
| 武器限制 | 刀/枪 (`blade`, `spear`) |
| CD | 4.8秒 |
| 内力消耗 | 18 |
| 基础倍率 | 1.6x |
| 命中次数 | 1次 |
| 成长 | 每级 +0.12 倍率 |
| 战斗表现 | 快速突刺，造成高额单体伤害并附带流血DOT |

#### 五雷掌 (`core_chain_lightning`)

| 参数 | 值 |
|------|-----|
| 名称 | 五雷掌 |
| 类型 | 远程连锁 |
| 武器限制 | 法球/杖 (`orb`, `staff`) |
| CD | 6.0秒 |
| 内力消耗 | 22 |
| 基础倍率 | 1.4x |
| 命中次数 | 1次（连锁另算） |
| 成长 | 每级 +0.1 倍率 |
| 战斗表现 | 掌心射出雷电，命中后连锁弹射至附近敌人 |

## 九、离线战斗规则

```
离线期间:
  - 正常计算战斗轮次和掉落
  - 凡品/精良 → 自动分解为锻材(上线时汇总显示)
  - 玄品 → 存入"待鉴定"队列(上线后手动查看)
  - 真意及以上 → 存入"出关所得·高光"(上线后逐个展示掉落动画)

上线时:
  "出关所得"弹窗 → 先显示总收益 → 再逐个播放高品质掉落的表演动画
```

## 十、战斗系统验收标准

1. 三个流派的核心武技均可正常释放并产生预期伤害
2. 战斗表演流畅，无明显卡顿
3. 掉落物品品质与节点/敌人类型匹配
4. 离线战斗产出与在线成比例
5. Boss战有明显的阶段感和视觉冲击力
6. 装备变更后，战斗表现有可感知的变化
