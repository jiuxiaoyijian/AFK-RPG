# 版本记录

> 武侠挂机 RPG — 项目重构版本管理

## 版本约定

- 采用 **语义化版本**：`vMAJOR.MINOR-tag`
- `v0.x` 系列为重构阶段，每个里程碑对应一个 minor 版本
- `v1.0-rc` 为首个发布候选，标志重构完成
- 每个版本发版后在本目录产出对应的 `vX.Y-tag_发版说明.md`
