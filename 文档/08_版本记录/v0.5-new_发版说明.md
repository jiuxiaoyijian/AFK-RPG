## 四、背包系统升级清单

| 功能 | 现状(v0.4) | 目标(v0.5) |
|------|-----------|-----------|
| 容量上限 | 无限 | 200 件（≥90%警告，≥100%强制整理弹窗） |
| 批量分解 | 无 | 选择品质阈值 → 一键分解未锁定装备 |
| 回收站 | 无 | 最近 30 条分解记录可恢复 |
| 自动分解 | 仅阈值 | 阈值 + 锁定豁免 + 传奇保护 |

## 五、新手引导脚本

### 强制序列（7 步）

| 步骤 | 触发 | 高亮 | 完成条件 |
|------|------|------|---------|
| welcome | 新游戏进入 | 战斗画面 | 等待 5 秒 |
| first_kill | 首次击杀 | 掉落物 | 击杀 3 个敌人 |
| open_inventory | 击杀 3 个后 | 导航栏背包按钮 | 打开背包 |
| equip_first | 打开背包后 | 最高品质装备 | 完成一次换装 |
| close_inventory | 换装后 | 关闭按钮 | 关闭背包 |
| show_nav | 关闭后 | 导航栏 | 等待 3 秒 |
| guide_complete | show_nav 完成 | 无 | 自动完成 |

### 软提示（5 类）

| 触发条件 | 提示内容 |
|---------|---------|
| 首次掉落橙装 | "传奇装备！打开背包看看它的传奇特效" |
| 首次遇到 Boss | "Boss 出现了！击败它可获保底传奇掉落" |
| 首次可萃取装备 | "你可以在百炼坊(B)中萃取传奇特效" |
| 首个钥石 | "你获得了秘境钥石！按 P 挑战秘境" |
| 首次每日目标 | "每日修行任务在右上角" |
