# 变更日志

## 格式说明

每条变更记录包含：
- **日期**：变更生效日期
- **类型**：新增 / 修改 / 删除 / 重构
- **内容**：变更的具体描述
- **负责人**：执行变更的成员
- **关联文档**：受影响的文档或系统
