## 当前里程碑：v0.2 — 核心玩法

### 目标

搭建完整的 Build 构建 → 战斗 → 掉落 → 强化 核心循环，使玩家可以体验到：
- 选择流派（核心技能）
- 装备词缀影响战斗数值
- 击杀掉落 → 拾取 → 对比 → 替换装备
- 基础的进阶系统（传承 / 百炼坊入口）

### 交付清单

- [ ] 装备生成器按品质 / 等级产出装备
- [ ] 词缀系统正确计算战斗属性
- [ ] 背包 UI 支持装备对比和替换
- [ ] 战斗 HUD 正确显示技能循环状态
- [ ] 至少 2 个章节可推进
- [ ] 存档 / 读档稳定
- [ ] GitHub Pages 可访问原型 Demo
