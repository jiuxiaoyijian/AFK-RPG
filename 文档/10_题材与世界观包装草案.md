# 题材与世界观包装草案

## 1. 文档定位

本文档用于为当前项目补上统一、可落地的产品外壳。

目标不是一次性写完完整剧情，而是先明确：

- 项目的题材基调是什么
- 玩家扮演谁
- 为什么这个世界天然适合“挂机刷图 + 掉落构筑 + 持续推进”
- 章节、敌人、Boss、美术和 UI 应该朝什么方向统一

它主要服务于：

- 章节命名
- 敌人与 Boss 包装
- 场景视觉方向
- UI 与图标风格
- 后续美术资源制作与外包需求
- README、商店页和对外展示文案
- `12_开发排期表.md` 中 `P1-1` 与 `P1-2` 的执行基线
- `13_反桃花源修真命名包装表.md` 的上位世界观说明

## 2. 一句话产品包装

这是一个发生在 `反桃花源修真` 世界观下的横版挂机刷装游戏。

玩家扮演一名误入伪仙境的 `外来修士`，在表面温润、内里吃人的洞天中不断探境、破局、夺机缘。你以为自己是在修仙，实际上是在一套用香火、祠庙和洞天秩序编织出来的假桃源里，夺回属于自己的道统与活路。

推荐对外一句话卖点：

- `你以为自己进了桃花源修仙，其实进的是一座会吃人的伪仙境。`

## 3. 题材方向建议

当前项目最适合的题材方向，不建议走通用仙侠爽文，也不建议回到旧废土方向，而建议明确采用：

- `反桃花源`
- `伪仙境`
- `香火秩序`
- `洞天异变`
- `修真包装下的刷装破局`

这条方向能同时兼容当前项目已经成型的结构：

- 章节推进
- 刷图掉落
- 装备词条构筑
- 传奇追踪
- 研究成长
- 离线收益

也能自然支撑后续产品化目标：

- 章节主题差异
- 敌人与 Boss 包装
- 结果卡和阶段事件高光
- 首批美术资源统一风格
- Steam 展示图和宣传语

## 4. 世界观基础设定

### 4.1 世界背景

世人都说桃源之外是乱世，桃源之内是清净仙乡。

可真正的桃源并不存在。现在的这片洞天，只是被香火、旧祠、仪轨和某种异化仙法维持出来的 `伪仙境`。

它有几层核心设定：

- 表面是山雾、灵田、古祠、纸灯和清修之地
- 内里是吞人气运、炼人香火、役人命数的秩序机器
- 越往深处走，越能看见这座洞天并不是让人飞升，而是要把人炼成供奉它继续运转的材料

玩家每次进入节点、重复刷图、追逐异宝，都不是机械重复，而是在不断试图拆穿这套秩序、夺走它的宝物、并让自己活着走得更深。

### 4.2 玩家身份

玩家扮演的是 `外来修士`。

可选近义表达包括：

- 异乡修士
- 误入洞天者
- 破妄行人

这个身份的好处是：

- 非常适合“反复探境刷装”
- 非常适合“不断换路数、调 build”
- 不需要重剧情也能解释反复进入同一区域
- 能自然承接机缘追踪、异闻录、悟道、闭关所得等系统

### 4.3 核心叙事驱动

主线不是写成长篇剧情，而是围绕三个很清晰的动机推进：

- 破开眼前伪装出来的清净秩序
- 在洞天中夺取异宝与真意，补全自己的道统
- 打开更深层探境，逼近这座伪仙境的真正核心

这套叙事非常适合挂机游戏，因为它天然支持：

- 章节推进
- 重复刷图
- 稀有掉落
- Boss 首杀
- 更高难区域循环

## 5. 主角包装建议

### 5.1 主角外观方向

建议主角视觉风格：

- 不是正统仙门弟子，更像误入异境的行脚修士
- 衣装以布料、护臂、符带、旧器为主
- 要有“行路感”和“破局感”，而不是单纯贵气
- 可以带少量异化、侵染或护身法器痕迹

关键词：

- 行脚袍
- 护臂
- 旧符
- 香灰痕
- 风痕 / 血纹 / 雷白流派差异

### 5.2 主角职业定位

不建议把主角包装成“天命救世主”，更适合写成：

- 外来修士
- 误入洞天者
- 破妄行人

这样更适合长期刷图和装备迭代，不会和挂机玩法的重复性冲突。

### 5.3 三个核心流派的包装映射

为了让技能、HUD 文案和美术资源统一，建议明确采用当前已经接入游戏内的三流派包装名。

#### 御风道

包装说明：

- 借风行刃，破阵开路
- 更强调回环、割裂、风痕扩散和近身持续压制

视觉关键词：

- 风痕
- 桃叶碎影
- 回环刃气

#### 血劫道

包装说明：

- 以伤换势，以血逼命
- 强调裂伤、断命线和处决感

视觉关键词：

- 朱砂
- 裂伤
- 断命符印

#### 五雷道

包装说明：

- 借雷击窍，引电行脉
- 强调引雷、连锁、爆点和术式流转

视觉关键词：

- 雷白
- 金符
- 连锁爆点

## 6. 阵营与敌人包装

为了让敌人不再只是“配置表条目”，建议先统一三层敌对势力。

### 6.1 洞天瘴生

代表：

- 桃瘴鼠
- 炽骨犬

设定：

- 被桃瘴、香火与洞天秽气侵染的兽类
- 数量多、行动快，压迫感来自不断逼近与围堵

视觉关键词：

- 干瘦
- 瘴痕
- 发亮裂口
- 不自然的温顺外表与突变攻击姿态

### 6.2 祠役守众

代表：

- 纸甲巡祠
- 守田傀
- 护炉傩兵

设定：

- 原本负责守祠、看田、执仪和巡境的役众或傀儡
- 仍在执行错误而残酷的洞天秩序

视觉关键词：

- 纸甲
- 木傀
- 傩面
- 仪轨痕迹

### 6.3 香火异修

代表：

- 香灰巫祝
- 引雷役修
- 祭炉监院

设定：

- 依附祠庙、仪轨和异化修行法生存的变质修士
- 掌握香灰、祠火、引雷和祭炼之法

视觉关键词：

- 法袍
- 符纸
- 香灰
- 祠火
- 异化修行器具

## 7. 当前章节包装建议

### 7.1 第一章：桃溪外渡

定位：

- 洞天边缘最温和、也最容易骗人放下戒心的入口区域
- 玩家第一次感受到“这里看起来像桃源，但不对劲”的地方

当前功能定位：

- 新手理解战斗、掉落与探境节奏
- 建立“伪仙境 + 役众秩序 + 初阶异宝”印象

视觉关键词：

- 木桥
- 渡口
- 村舍
- 山雾
- 纸灯

Boss 包装：

- `伏祠山君`
- 它不是山野神灵，而是被供奉、被束缚、又被这片秩序役使的祠前异兽

### 7.2 第二章：祠下灵田

定位：

- 古祠之下、灵田之间，开始显露这套香火秩序如何运作
- 玩家从“可疑的桃源表象”真正进入“吃人的运行机制”

当前功能定位：

- 把敌人从洞天瘴生推进到更明确的役众、祠兵和异修
- 强化“香火 / 祠庙 / 灵田 / 祭炼”世界印象

视觉关键词：

- 灵田
- 古祠
- 井台
- 幡旗
- 香火雾

Boss 包装：

- `祭炉监院`
- 负责维持祭炉与役众秩序的监者，是玩家第一次正面对上“洞天如何吃人”的管理层象征

### 7.3 后续章节方向建议

后续章节建议继续沿“洞天边缘 -> 深层异变 -> 核心祭炼之地”推进。

可扩展方向：

- `雾隐丹井`
- `白骨洞天`
- `升仙古渡`
- `祠后禁园`
- `归真祭台`

这样既能保持主线递进，也能让每章拥有鲜明的视觉主题和异化层次。

## 8. 当前游戏内接入基线

为了避免文档、美术和程序再次分叉，当前应默认以下命名已经是项目正式口径：

- 玩家身份默认写作 `外来修士`
- 第一章默认写作 `桃溪外渡`
- 第二章默认写作 `祠下灵田`
- 第一章 Boss 默认写作 `伏祠山君`
- 第二章 Boss 默认写作 `祭炉监院`
- 三流派默认写作 `御风道 / 血劫道 / 五雷道`
- 高频系统名默认写作 `今日机缘 / 悟道 / 异闻录 / 机缘推演 / 闭关所得`
- 高频资源名默认写作 `香火钱 / 祠灰 / 灵核 / 真意残片`

后续如果没有特别说明，美术、外包、README、宣传图文和程序实现都应以这套口径为准。

## 9. UI 与美术风格方向

### 8.1 UI 风格建议

建议 UI 不走轻松卡通，也不走传统仙侠手游金光大边框，而应采用：

- 古意修真终端感
- 祠庙器物与文牍结合
- 清净外表下带异样裂痕的界面语言

关键词：

- 深青灰底
- 米白纸感
- 金橙 / 朱砂 / 雷白高亮
- 木框、纸符、切角、细描边
- 干净但不完全温和

### 8.2 稀有度视觉建议

- 普通：灰白
- 稀有：青蓝
- 史诗：紫
- 传奇：金橙
- 远古：暗金红

这些颜色应同步用于：

- 装备名
- 掉落提示
- 结果卡边框
- 图标描边

### 8.3 图标资源方向

图标建议保持：

- 武器与法器轮廓清晰
- 技能图标有流派辨识
- 资源图标具备香火、祠灰、灵核、残片感
- 结果卡能一眼区分掉落类别、章节事件与成长事件

## 10. 音频风格建议

为了和当前题材统一，音频建议走：

- 风声、木响、铃响、纸符、闷雷
- 低频不安氛围
- 祠火燃烧与远处祭礼残响
- 高光时刻以金石、雷裂和异样回声强化

不同场景可参考：

- 桃溪外渡：山风、水声、纸灯、轻微异响
- 祠下灵田：虫鸣、井声、香火、远处木鱼和不祥诵念
- Boss 战：更强低频、异样礼乐、祠火爆鸣与雷震

## 12. 当前结论

当前项目最适合的正式包装方向，不是废土，也不是泛化修仙，而是：

- `反桃花源`
- `伪仙境`
- `香火秩序`
- `探境破局`
- `夺异宝、补道统、拆假桃源`

这条路线能够最大限度兼容现有章节、敌人、Boss、技能和掉落系统，并天然适合：

- 刷装
- 稀有度分层
- 构筑成长
- 章节推进
- 结果高光

如果后续没有更强理由，建议将这一方向作为项目正式包装的默认基调，并以 `13_反桃花源修真命名包装表.md` 作为具体接入时的命名基线。
