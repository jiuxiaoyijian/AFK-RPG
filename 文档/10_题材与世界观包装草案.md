# 题材与世界观包装草案

## 1. 文档定位

本文档用于为当前项目补上更有传播性的题材外壳。

本轮不再沿用 `灾后黑铁废土` 方向，而正式切换为：

- `武侠 / 修仙`
- `反桃花源`
- `伪仙境`
- `美好外壳下的吞噬性秩序`

目标不是一口气写完完整剧情，而是先明确：

- 项目的题材基调是什么
- 玩家扮演谁
- 这个世界为什么一直要刷图、求宝、破局
- 章节、敌人、Boss、美术和 UI 应该朝什么方向统一

它应该服务于：

- 章节命名
- 敌人与 Boss 包装
- 场景视觉方向
- UI 与图标风格
- 后续美术资源采买和外包需求
- 游戏内文案替换

## 2. 一句话产品包装

这是一个发生在 `伪桃源洞天` 中的桌面挂机刷装游戏。

玩家扮演一名误入桃源的 `外来修士`。表面上这里云深花盛、村舍安宁、人人劝你留下修行；实际上整座桃源是一处会吞人、会养蛊、会把修士与乡民一同炼成香火资粮的假仙境。

你要做的不是单纯飞升，而是：

- 在这个会吃人的桃源里活下来
- 不断刷取机缘、异宝、禁术与道统残篇
- 重组自己的修行 build
- 拆穿桃源秩序
- 向更深处的洞天与祠庙推进

## 3. 题材方向建议

当前最适合项目现状的题材方向，不建议走常规“纯爽文修仙”，而建议采用：

- `反桃花源修真`
- `伪仙境`
- `山水表象 + 内里异化`
- `人间温柔表层下的香火、献祭、吞噬秩序`

这个方向的好处很明确：

- 有一句话就能讲清的传播点
- 在中文语境里有辨识度
- 适合挂机游戏的“表面平静、底层异化”气质
- 方便把当前系统自然换皮，而不是推翻重做

它能兼容当前已有玩法骨架：

- 自动推进战斗
- 机缘掉落与构筑
- 图鉴追踪与推荐刷图
- 研究树与长期成长
- 阶段事件与结果卡

## 4. 世界观基础设定

### 4.1 世界背景

世人都说 `桃源洞天` 是一处远离尘世、可避兵祸、可证长生的福地。

凡人入内，会看见：

- 桃林
- 渡口
- 村舍
- 灵田
- 古祠
- 清修之士

修士入内，会得到：

- 灵气
- 丹法
- 残卷
- 法宝
- 供奉

但真相是，这座桃源并不通向超脱，而是一套自我循环的 `伪飞升秩序`：

- 村民并非真正安居，而是被香火与阵法驯化的壳
- 山川灵气并非自然流转，而是从外来者身上倒抽而来
- 祠庙、井、桥、渡口、桃树全是阵眼的一部分
- 所谓“修行有成”，本质上是在一步步把自己炼进桃源

整个世界观的关键，不是“我要成仙”，而是：

- 我要看破它
- 我要利用它
- 但不能被它吞下去

### 4.2 玩家身份

玩家扮演的是 `误入桃源的外来修士`，可以理解为：

- 云游散修
- 被逐出山门的外门弟子
- 为避兵祸而误入山中的异乡客
- 被机缘引入洞天、又察觉不对劲的求道者

这个身份的好处：

- 非常适合“刷图拿宝”
- 非常适合“词条重组 build”
- 不需要复杂大剧情就能解释反复进入同一区域
- 能自然承接机缘、追踪、图鉴、悟道、闭关等系统

### 4.3 核心叙事驱动

主线不是拯救天下，而是明确的求生与破局动机：

- 识别桃源中的伪善与阵眼
- 获取残篇、异宝与禁术以强化自身
- 打穿不同层级的伪仙秩序
- 找到真正操盘整座桃源的“庙、主、阵、道”
- 最终决定是逃出桃源、夺走桃源，还是反证其道

这套结构非常适合挂机游戏，因为它天然允许：

- 章节推进
- 重复刷图
- 稀有掉落
- Boss 首杀
- 更高层洞天循环

## 5. 主角包装建议

### 5.1 主角外观方向

建议主角视觉风格：

- 并非正统仙门贵公子
- 更像 `风尘散修 + 山野行路人`
- 衣装有旅修感、旧道袍感、外来者感
- 身上开始逐渐沾染桃源气息与禁术痕迹

关键词：

- 旧青衫
- 披风
- 折剑 / 法器残件
- 护符与符纸
- 桃木、铜铃、旧册、香灰痕

### 5.2 主角职业定位

不建议把主角写成“天命之子”，而建议写成：

- 异乡修士
- 山野散修
- 误入洞天者
- 破妄行人

这样更利于长期刷图和装备迭代，也不会和挂机重复性产生叙事冲突。

### 5.3 三个核心流派的包装映射

为了让技能和世界观统一，建议给现有三流派补上产品化称呼。

#### 旋风流

建议包装名：

- `御风道`

包装说明：

- 借风行刃，破阵开路
- 更像散修在山路与庙廊之间练出来的实战道法

视觉关键词：

- 青白风痕
- 桃叶乱卷
- 刃气回环

#### 流血流

建议包装名：

- `血劫道`

包装说明：

- 以伤换势、以血逼命
- 像是桃源禁术里最容易见效、也最容易失控的一支旁门

视觉关键词：

- 朱砂血线
- 裂帛伤痕
- 处决符印

#### 雷链流

建议包装名：

- `五雷道`

包装说明：

- 借雷击窍，引电行脉
- 看似正道雷法，实则在桃源中也被异化成更妖异的连锁术

视觉关键词：

- 雷白电痕
- 金符灼光
- 连锁爆点

## 6. 阵营与敌人包装

为了让当前敌人不再只是“数据条目”，建议先划分三个阵营层。

### 6.1 桃源伪民

代表：

- 纸衣乡民
- 笑面迎客
- 守桥老仆

设定：

- 他们看起来像安宁村民，言语温和，行为有礼
- 但本质上已经被香火与阵法驯化，只剩维持桃源秩序的壳

视觉关键词：

- 木讷笑容
- 纸灯笼
- 桃木饰物
- 衣摆整齐却毫无生气

### 6.2 山野异化物

代表：

- 桃瘴鼠
- 枯骨犬
- 井蛭

设定：

- 长期被桃源灵气和香火秽气浸染后的山野生灵
- 数量多、速度快、压迫感来自“本该自然，却越来越不对劲”

视觉关键词：

- 毛皮腐粉
- 枯枝骨刺
- 花瓣与腐肉并存
- 眼中泛香火红光

### 6.3 祠庙执事与伪仙守卫

代表：

- 巡祠力士
- 执灯道人
- 镇井傀儡
- 香火护法

设定：

- 他们是维持洞天运转的执行者
- 看似修行者，实则多数只是被祠庙与阵法借壳驱动的“守序器官”

视觉关键词：

- 道袍残旧
- 祠器与锁链
- 木面具
- 金漆、灰烬、桃木与血线

## 7. 当前章节包装建议

### 7.1 第一章：桃溪外渡

定位：

- 误入桃源后的第一层安全假象
- 有桥、有渡口、有茶摊、有村舍，看起来最像“人间好去处”

当前功能定位：

- 新手入口
- 玩家第一次理解“这里不对劲”
- 敌人与环境的异样感应从细节里慢慢浮出来

可承载内容：

- 纸衣乡民
- 桃瘴鼠
- 守桥老仆
- 第一位真正意义上的“迎客者”Boss

视觉关键词：

- 水雾
- 桃花
- 渡船
- 旧桥
- 灯火温暖却过于整齐

### 7.2 第二章：祠下灵田

定位：

- 桃源真正露出内里结构的地方
- 灵田、祠堂、香火与供奉开始连成完整秩序

当前功能定位：

- 中期推进区
- 玩家开始理解“这里的修行与生长，都是拿别的东西换来的”

可承载内容：

- 巡祠力士
- 井蛭
- 执灯道人
- 祠下主事者 Boss

视觉关键词：

- 灵田
- 古祠
- 香灰
- 供桌
- 桃木牌位

## 8. UI 与系统文案方向

为了让题材统一，建议后续将系统文案整体向以下词汇靠拢。

### 8.1 系统层替换建议

- `章节推进` -> `探境`
- `研究树` -> `悟道`
- `离线收益` -> `闭关所得`
- `每日目标` -> `今日机缘`
- `图鉴` -> `异闻录`
- `掉落统计` -> `机缘推演`
- `传奇词条` -> `异宝 / 禁术真意`
- `build` -> `道统 / 路数`

### 8.2 结果卡与提示语气

提示文案不建议走热血工业风，而建议采用：

- 表面平静
- 古意克制
- 夹带不安

例如：

- “你已深入桃溪外渡。”
- “此物似有旧主，不宜久持。”
- “祠中香火更盛，说明你离真相更近了。”
- “桃源愿意给你更多，通常不是好事。”

## 9. 视觉方向建议

### 9.1 基础视觉气质

不是纯黑暗，也不是纯仙气，而是：

- 越好看，越不对劲
- 越安静，越让人不安
- 越像桃源，越像陷阱

### 9.2 色彩建议

- 主色：雾青、旧白、桃粉、墨青
- 高光：血朱、香火金、雷白
- 禁忌高光：暗红、枯金、幽绿

### 9.3 美术关键词

- 桃花
- 古桥
- 祠堂
- 井
- 山雾
- 符纸
- 木偶
- 纸人
- 骨灯
- 香炉

## 10. 当前结论

当前最适合项目下一阶段产品包装的题材，不再是：

- `灾后黑铁废土`

而是：

- `反桃花源修真`

更准确地说，是：

- `表面像世外桃源，内里却是吃人洞天的伪仙境修真题材`

这条路线的优势是：

- 更适合中文语境传播
- 概念钩子更强
- 更容易做出一句话卖点
- 能自然承接当前挂机、刷装、构筑、阶段事件和长期成长系统

后续如果继续做 `P1-1` 接入，优先级应是：

1. 统一章节、敌人、Boss 命名
2. 统一 HUD / 结果卡 / 系统面板常用文案
3. 让“反桃花源”概念在 UI、章节和事件里立即被玩家感知
