# 题材与世界观包装草案

## 1. 文档定位

本文档用于为当前项目补上“产品外壳”。

目标不是一口气写完完整剧情，而是先明确：

- 项目的题材基调是什么
- 玩家扮演谁
- 这个世界为什么一直在战斗和刷装
- 章节、敌人、Boss、美术和 UI 应该朝什么方向统一

它应该服务于：

- 章节命名
- 敌人与 Boss 包装
- 场景视觉方向
- UI 与图标风格
- 后续美术资源采买和外包需求

## 2. 一句话产品包装

这是一个发生在 `灾后黑铁废土` 上的桌面挂机刷装游戏。

玩家扮演一名在废墟边境不断回收旧时代兵装、猎杀异化怪物、突破封锁带的 `边境回收者`，通过收集核心兵装和古代特效词条，不断重组自己的战斗方式，向更深层的禁区推进。

## 3. 题材方向建议

当前最适合项目现状的题材方向，不建议走纯奇幻，也不建议走纯现代枪械，而建议采用：

- `黑暗废土`
- `灾后工业遗迹`
- `古代熔炉科技残骸`
- `异化生物 + 失控守卫`

这样能同时兼容当前已有内容：

- 废墟边境
- 焦灼矿场
- 骨甲卫兵
- 疫病萨满
- 锻炉圣卫
- 熔核监工
- 旋风 / 流血 / 雷链三类技能风格

也更适合后续做：

- 装备掉落与词条构筑
- 稀有度分层
- 章节推进与禁区解锁
- 机械与神秘混合的视觉资源

## 4. 世界观基础设定

## 4.1 世界背景

旧时代曾经依赖一套名为 `熔核炉心` 的巨型能源体系，支撑了工业、兵装和城市防御。

后来炉心网络在一场被称为 `余烬断潮` 的灾变中失控：

- 城市防线崩溃
- 矿场和铸造设施被永久高温侵蚀
- 大量守卫兵装失控
- 生物发生异化
- 遗迹中残留了大量可被重新拼装的兵装模块与传奇特效

如今文明退缩到边境聚落，只能靠回收者不断进入废墟区与熔灾区，回收旧时代装备与核心能源，以维持防线。

## 4.2 玩家身份

玩家扮演的是 `边境回收者`，定位可以理解为：

- 废土猎兵
- 遗迹回收者
- 临时佣兵
- 炉心遗产的拼装者

这个身份有几个好处：

- 非常适合“刷图拿装备”
- 非常适合“词条重组 build”
- 不需要复杂剧情就能解释反复进入同一区域
- 能自然承接任务、追踪、图鉴、研究等系统

## 4.3 核心叙事驱动

主线不是宏大剧情，而是明确的推进动机：

- 回收旧时代兵装和炉心碎片
- 清理边境威胁
- 打开被封锁的更深层禁区
- 找到灾变源头与失控炉心

这套叙事结构适合挂机游戏，因为它允许：

- 章节推进
- 重复刷图
- 稀有掉落
- Boss 首杀
- 更高难区域循环

## 5. 主角包装建议

## 5.1 主角外观方向

建议主角视觉风格：

- 轻甲或中甲废土战士
- 身上有回收拼装感
- 核心武器与炉心背载装置可见
- 兼具工业感与冷兵器/能量武器感

关键词：

- 风衣残片
- 护肩
- 旧时代兵装部件
- 炉光纹路
- 金属与布料混搭

## 5.2 主角职业定位

不建议把主角写成“救世主”，而建议写成：

- 边境回收者
- 炉心猎兵
- 断潮清剿者

这样更利于长期刷图和装备迭代，不会和挂机重复性产生叙事冲突。

## 5.3 三个核心流派的包装映射

为了让技能和世界观统一，建议给现有三流派补上产品化称呼。

### 旋风流

建议包装名：

- `裂风回旋`

包装说明：

- 依靠高转速近战兵装切开敌阵
- 属于旧时代的近距破阵战法

视觉关键词：

- 金属风压
- 裂刃残影
- 高速切削火花

### 流血流

建议包装名：

- `猩红切割`

包装说明：

- 通过连环割裂与处决技术让目标持续失血
- 更像回收者在废土中发展出的高效狩猎术

视觉关键词：

- 伤口裂痕
- 暗红血线
- 处决印记

### 雷链流

建议包装名：

- `炉闪导链`

包装说明：

- 以炉心残能驱动连锁电流
- 擅长处理中远距离与密集群体

视觉关键词：

- 蓝白电弧
- 炉光导线
- 连锁爆点

## 6. 阵营与敌人包装

为了让当前敌人不再只是“数据条目”，建议先划分三个阵营层。

## 6.1 边境异化兽群

代表：

- 腐鼠
- 爆燃猎犬

设定：

- 灾变后被炉尘、毒雾和高热污染的变异生物
- 数量多、速度快、压迫感来自群体性

视觉关键词：

- 干瘦
- 炽红眼
- 溃烂皮毛
- 发热裂口

## 6.2 旧时代守卫残军

代表：

- 骨甲卫兵
- 采场守卫
- 锻炉圣卫

设定：

- 原本用于守卫矿区、城防和熔炉设施的装甲兵装
- 灾变后仍维持错误的守卫指令

视觉关键词：

- 沉重装甲
- 古旧金属
- 冷白或暗红能量缝隙
- 仪式化或工业化头盔造型

## 6.3 熔灾信徒与异端工匠

代表：

- 疫病萨满
- 电链驯导者
- 熔核监工

设定：

- 在灾变后依附炉心残能生存的一批异端幸存者或变质管理者
- 他们掌握污染、导能和熔核控制技术

视觉关键词：

- 熔痕法袍
- 导能背架
- 炉芯面具
- 电火与灰烬并存

## 7. 当前章节包装建议

## 7.1 第一章：废墟边境

定位：

- 文明边界外的旧城废区
- 是回收者最早进入的封锁地带

当前功能定位：

- 新手理解战斗与掉落系统
- 建立边境异化兽群和失控守卫的基础印象

视觉关键词：

- 倒塌路障
- 断裂墙体
- 骨铁废料
- 灰绿雾气
- 暗沉金属

Boss 包装：

- `铁爪巨兽`
- 可设定为被旧时代改造过的重型实验兽
- 是“边境封锁带”的看门者

## 7.2 第二章：焦灼矿场

定位：

- 灾变后持续发热的熔矿区
- 炉心失控最早扩散的区域之一

当前功能定位：

- 把敌人从废墟生物推进到更强的矿区守卫和熔灾信徒
- 强化“熔核科技 + 高热废土”的世界印象

视觉关键词：

- 炽热矿道
- 熔岩裂缝
- 铁轨残骸
- 蒸汽阀门
- 橙红高温光

Boss 包装：

- `熔核监工`
- 可设定为矿场自动监管核心的人形化失控单位
- 是“从工业灾难进入更深层禁区”的转折点

## 7.3 后续章节方向建议

后续章节建议继续沿“边境 -> 深区 -> 炉心禁区”推进。

可扩展方向：

- `风蚀高墙`
- `电骸工坊`
- `黑井回廊`
- `余烬圣所`
- `断潮炉城`

这样既能保持工业废土主线，又能让每章拥有不同视觉主题。

## 8. UI 与美术风格方向

## 8.1 UI 风格建议

建议 UI 不走轻松卡通，而走：

- 工业仪表盘
- 废土终端界面
- 炉心扫描器

关键词：

- 深灰底
- 锈红 / 炉橙 / 冷蓝高亮
- 品质边框
- 矩形切角
- 金属卡扣和数据屏感

## 8.2 稀有度视觉建议

- 普通：冷灰
- 稀有：蓝
- 史诗：紫
- 传奇：金橙
- 远古：暗金红

这些颜色应同步用于：

- 装备名
- 掉落提示
- 结果卡边框
- 图标描边

## 8.3 图标资源方向

图标建议保持：

- 武器与零件轮廓清晰
- 技能图标有流派辨识
- 资源图标具备工业/炉心感
- 结果卡能一眼区分掉落类别、章节事件与成长事件

## 9. 音频风格建议

为了和题材统一，音频也建议走：

- 低频工业环境
- 金属碰撞
- 炉火喷发
- 电流爆裂
- 失真警报

不同场景：

- 边境废墟：压抑、风沙、残响
- 焦灼矿场：高热、链条、蒸汽、熔岩鼓动
- Boss 战：更强低频和金属警示音

## 10. 对后续开发的实际帮助

这份包装草案不是装饰物，它应直接服务后续开发决策：

### 10.1 服务功能命名

- 章节名
- 流派名
- Boss 名
- 传奇特效名
- UI 标题文案

### 10.2 服务美术外包

- 给角色、敌人、场景、图标确定统一方向
- 避免后续资源风格严重割裂

### 10.3 服务 Steam 包装

- 未来商店页截图、标题包装、宣传语和主视觉都需要一个统一题材框架

## 11. 当前结论

当前项目最适合的产品包装方向，不是轻奇幻，也不是纯现代，而是：

- `灾后黑铁废土`
- `工业遗迹`
- `熔核科技残骸`
- `异化生物与失控守卫`

这条路线能够最大限度兼容现有章节、敌人、Boss、技能和掉落系统，并且天然适合：

- 刷装
- 稀有度
- 构筑
- 禁区推进
- 章节视觉差异

如果后续没有更强理由，建议将这一方向作为项目正式包装的默认基调。
