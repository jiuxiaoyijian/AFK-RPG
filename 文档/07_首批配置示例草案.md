# 首批配置示例草案

本文件提供首版 MVP 可直接使用的配置样例草案。目标不是最终平衡，而是让其他 agent 可以马上创建：

- 初始技能
- 初始敌人
- 初始章节
- 初始装备词条
- 初始掉落表

建议先按此草案跑通，再做数值调优。

## 1. 首发 3 个流派

MVP 只做以下 3 个流派：

- `whirlwind` 旋风流
- `bleed` 流血流
- `chain_lightning` 雷连锁流
