# 美术资源清单

> 武侠江湖风格，水墨+写意为主基调。
> 优先级对齐开发路线图（P0→P1→P2→P3）。

---

## 一、美术风格定义

| 维度 | 定义 |
|------|------|
| 整体风格 | 水墨武侠，淡雅留白 |
| 配色基调 | 深灰/墨色底 + 金色/朱红点缀 |
| 线条风格 | 简笔写意，非精细写实 |
| 参考方向 | 太吾绘卷/江湖悠悠/部分武侠独立游戏 |
| 禁忌 | 不要修仙飞升元素（仙鹤、祥云、飞剑） |
