# AFK-RPG 潮流新国风 Q 版美术规范 v1

日期：2026-03-29

---

## 1. 风格结论

`AFK-RPG` 的正式美术方向调整为：

- `潮流新国风武侠`
- `轻旅行感江湖`
- `2.2 - 2.5 头身 Q 版角色`

这套风格服务的不是“沉重黑暗武侠”，而是：

- 长时间挂机观看依然舒适
- 角色与场景具备更强收藏感
- 战斗与刷装仍保留明确的流派识别度
- 整体气质更接近“风景旅途中的江湖冒险”

一句话定义：

`AFK-RPG` 采用 2.2 - 2.5 头身的潮流新国风武侠风格，以明快、干净、可长期挂机观看的江湖旅途场景为背景，用清晰的武器剪影、流派配色和轻量化国风器物感，承载挂机刷装、武学 Build 与重入江湖的长期成长体验。

---

## 2. 风格目标

本项目不再追求以下方向：

- 暗黑压抑的 grimdark 武侠
- 过于写实的苦大仇深江湖
- 老式页游式厚重黑金古风
- 低幼婴儿感的纯萌系二头身

本项目要追求的是：

- 年轻化
- 可爱但不幼稚
- 风景感、旅游感、探索感
- 轻松但仍然有武侠识别度
- 适合 Steam 玩家接受的强风格化独立游戏气质

---

## 3. 角色规范

### 3.1 头身与比例

- 角色比例固定在 `2.2 - 2.5 头身`
- 不做严格 2 头身圆滚滚幼儿比例
- 头部略大，但腿部和兵器仍需保留“侠客感”
- 躯干与四肢可适度缩短，但动作轮廓必须利落

### 3.2 角色气质

- 年龄感：青年侠客
- 表情：冷静、专注、略带松弛感
- 情绪：出发、行旅、应战，而非愤怒、阴郁、仇杀
- 避免“大眼幼态”“卖萌表情”“儿童化站姿”

### 3.3 服装与配件

- 国风服装结构需保留：
  - 交领
  - 披肩或短披风
  - 腰封
  - 护腕
  - 靴具
- 复杂层数控制在 `2 - 3 层`
- 避免过多飘带、仙侠发光纹样、重度首饰堆砌
- 材质优先：
  - 棉布
  - 麻布
  - 皮革
  - 木柄
  - 铜件

### 3.4 武器表达

- 武器必须服务“流派差异”
- 允许适度夸张比例，前提是剪影清晰
- 推荐：
  - 御风：轻刀、短刃、流线型刀势
  - 血劫：重刀、裂刃、断口、钩刃
  - 五雷：符刃、短杖、带金属或玉石节点的法器类兵器

### 3.5 角色剪影

- 优先级高于面部细节
- 远距离一眼应能看出：
  - 这是侠客
  - 使用什么武器
  - 属于什么路数
- 角色设计必须先看缩略图，再看大图

## 5. 场景规范

### 5.1 场景总基调

- `江湖旅行模拟器`
- `白天或黄昏的清爽中国古风场景`
- `可长期挂机观看`

场景应给玩家的第一感觉是：

- 想继续往前走
- 想看看下一个地方
- 这是江湖旅途，而不是深渊地牢

### 5.2 推荐场景类型

- 渡口
- 山门
- 集市外街
- 客栈门前
- 栈道
- 山路
- 城郊驿站
- 竹林小径
- 货栈码头

### 5.3 构图规则

- 中间必须留出清晰战斗走廊
- 前景点缀：
  - 灯笼
  - 栏杆
  - 旗幡
  - 摊位边角
- 中景承担主要建筑和道路识别
- 远景负责楼阁、山体、桥梁、塔影
- 不允许全画面细节平均铺满

### 5.5 横版三层背景生产规则

当前项目的战斗场景是：

- 主角固定在左侧三分之一
- 怪物从右侧刷出
- 背景存在 `far / mid / near` 三层视差

因此正式背景生产**不建议**采用“先生成 1 张完整背景，再机械拆成 3 层”作为最终方案。

正确做法应为：

- 先锁同一场景设定
- 再分别生成：
  - `far` 远景层
  - `mid` 中景主场景层
  - `near` 前景遮挡层

原因：

- 视差层不是亮度分层，而是语义分层
- 远景应承担山体、塔楼、天光、远村、桥影
- 中景应承担主要战斗舞台和核心建筑识别
- 前景应承担栏杆、旗幡、树枝、摊位边角、石块等推进遮挡物

#### 当前项目中的临时方案

目前仓库中的：

- `tools/generate_parallax_background_layers.py`

属于“验证型快速方案”，它会把同一张背景图进行后处理，生成：

- `__far`
- `__mid`
- `__near`

优点：

- 快
- 可验证视差逻辑
- 适合早期跑通功能

缺点：

- 前景不是独立构图
- 中景和远景缺少真正的语义分离
- 不适合长期作为正式量产方式

#### 正式量产建议

正式资源建议按以下流程生产：

1. 先生成 1 张“场景母版 / 构图参考”
2. 基于同一构图和色板，分别生成 `far / mid / near`
3. 最后再导入游戏并做合成验证

也就是说：

- 不是“1 张图硬拆三份”
- 也不是“完全独立随机生成三张互不相关的图”
- 而是“同一场景设定下，协同生成三张分层图”

#### 三层职责

`far`

- 最少细节
- 最远轮廓
- 颜色最淡
- 只负责氛围和纵深

`mid`

- 主要可读层
- 决定当前关卡的地点识别
- 必须保证中央战斗走廊清晰

`near`

- 细节最少但遮挡最强
- 只放少量前景器物
- 用于增强“不断向前推进”的体感

#### 下一阶段的标准

从第二轮风格样图开始，正式场景量产默认采用：

- `同场景三张协同生成`

不再默认采用单图后处理拆层作为正式资源来源。

### 5.4 颜色规范

推荐基础色：

- 米白：`#E9E2D3`
- 雾灰：`#D5D1C8`
- 青灰：`#AAB7B2`
- 茶褐：`#8B6B4A`
- 橙金：`#D9A24A`
- 松绿：`#5F8F69`
- 靛青：`#4C668A`
- 墨棕：`#4E4035`

使用原则：

- 高明度、中低饱和
- 局部暖色点亮，不全图发黄
- 不要全图黑灰压死
- 不要荧光国潮撞色

## 6. UI 规范

### 6.1 总体方向

- `轻国风器物感`
- `干净、薄、清爽`
- `比厚重暗黑 UI 更明亮，但不能廉价`

### 6.2 材质建议

- 浅木
- 旧铜
- 温润石料
- 磨砂青黑底
- 少量宣纸或布纹辅助

### 6.3 结构原则

- 尽量用小而清楚的模块
- 少做大面积黑幕遮罩
- 避免页游式书卷、木牌、重金边厚边框堆叠
- 战斗区域优先级永远高于 HUD

### 6.4 图标方向

- 扁平化 `70%`
- 手绘感 `30%`
- 小尺寸优先保证识别度
- 不追求写实，不做复杂金属高光

---

## 7. 特效规范

### 7.1 总原则

- 特效短、亮、清
- 强调利落，不强调压迫
- 用“旅行中的轻快武学”替代“沉重黑暗的毁灭感”

### 7.2 各类效果

- 刀风：弧线、薄片、短拖尾
- 雷击：亮边、折线、短促爆发
- 血劫：赤色冲击、裂纹感，不做脏污血浆
- 掉落：金光、符纸、器物辉光
- 结算：旅途达成感、奇遇感、收获感

## 8. 明确禁止项

- grimdark
- horror
- heavy gore
- photorealistic anatomy
- baby-like proportions
- giant anime eyes
- xianxia flying swords
- cloud sea immortals
- dark dungeon as default backdrop
- mobile splash-art over-rendering
- 低幼表情包式二头身
- 仙侠手游海报式高饱和国潮

## 10. 执行建议

建议后续美术执行采用以下顺序：

1. 先锁角色比例和脸型语言
2. 再锁场景色板和旅游感氛围
3. 再统一 UI 材质与图标简化程度
4. 最后再决定哪些系统保留更强的武侠危险感

不建议：

- 一边继续做暗黑武侠资源，一边又尝试 Q 版资源
- 先大量出图，再回头统一角色比例
- 让角色、场景、UI 走三种不同国风语义

## 11. 当前结论

最终美术方向已从：

- `半写实新国风暗黑武侠`

调整为：

- `潮流新国风 Q 版武侠`

主张：

- 更亮
- 更轻
- 更年轻
- 更适合挂机长期观看
- 更接近 Steam 独立游戏用户可接受的强风格化审美

保留：

- 武侠识别度
- 流派差异
- 江湖旅行感
- 装备与武学成长感
