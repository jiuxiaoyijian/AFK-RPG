# 反桃花源修真命名包装表

## 1. 文档定位

本文档用于把 `反桃花源修真` 方向进一步拆成可直接用于策划、文案、美术和程序接入的命名表。

它主要服务四件事：

- 统一章节、敌人、Boss、资源和 UI 文案口径
- 避免后续接入时又回到泛化修仙或旧废土词汇
- 给美术、外包和宣传文案提供稳定的关键词
- 为下一轮游戏内文本替换提供第一版基准

## 2. 命名总原则

- 表面温润，内里不安
- 先有古意，再露异化
- 不走流水线爽文修仙名词堆砌
- 不写得太黑话，要保证玩家一眼能懂
- 同一套词汇要同时支撑 `挂机刷装`、`构筑成长` 和 `章节破局`

## 3. 一句话卖点与传播钩子

推荐一句话卖点：

- `你以为自己进了桃花源修仙，其实进的是一座会吃人的伪仙境。`

可选传播钩子：

- `这不是世外桃源，这是吃人的洞天。`
- `你在修仙，也在被整座桃源拿去修。`
- `越往里修，越像在把自己炼成香火。`

## 4. 玩家身份与主循环称呼

推荐主身份：

- `外来修士`

可选替代：

- `异乡修士`
- `误入洞天者`
- `破妄行人`

推荐主循环表达：

- `探境`
- `破局`
- `求机缘`
- `夺异宝`

## 10. 结果卡与事件文案建议

推荐标题体系：

- `破局`
- `首诛`
- `悟道有成`
- `异宝现世`
- `机缘已至`

示例：

- Boss 首杀：`首诛 伏祠山君`
- 新章节解锁：`探境已开：祠下灵田`
- 研究升级：`悟道有成：战斗训练`
- 传奇首次获得：`异宝现世`
- 每日目标完成：`今日机缘已成`

## 11. 当前接入建议

推荐按下面顺序进入游戏内：

1. 先替换章节名、面板标题和 DailyGoal/离线收益等高频 UI 文案
2. 再替换敌人、Boss、掉落和结果卡文案
3. 最后统一资源图标、美术关键词和 README/对外展示文案

这样做的原因是：

- 改动最可控
- 玩家最容易先感知到变化
- 不会一上来就被全量文本替换拖慢实现节奏
