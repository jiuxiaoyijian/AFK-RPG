# AFK-RPG Public Demo

当前版本定位为 `公开试玩版`，重点验证：

- 自动战斗与刷装主循环是否清晰
- 背包、成长中心、推演与秘境这三类二级面板是否易懂
- Web 版在桌面浏览器中的可玩性与稳定性

## 操作说明

- `开始试玩 / 继续 / 清空档位`：在启动菜单选择
- `I`：背包
- `B`：百炼坊
- `U`：成长中心
- `O`：异闻录
- `P`：推演与秘境
- `F5`：快速存档
- `F8`：快速读档
- `Esc`：关闭当前弹窗
